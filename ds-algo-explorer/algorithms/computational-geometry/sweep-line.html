Sweep Line Algorithm

Measure total union length of intervals on a number line using left-right sweep.

Sweep line processes events (interval start/end) sorted by x-coordinate. Maintain a count of active intervals. When count transitions 0→1 (start of covered region), record left boundary. When it transitions 1→0, add (current_x - left) to total union length. O(n log n) sorting, O(n) sweep. Generalizes to 2D rectangles, area union, etc.

Red dashed line = sweep position. Green segments = covered region. Blue = active intervals. The total union length accumulates as sweep proceeds.
