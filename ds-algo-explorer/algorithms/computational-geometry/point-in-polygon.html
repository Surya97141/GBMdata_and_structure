Point in Polygon

Ray casting algorithm — count ray crossings, O(n) per query.

Ray casting: cast a horizontal ray from the query point to the right (+x direction). Count how many polygon edges the ray crosses. If odd → inside. If even → outside. Edge cases: ray through vertex is handled by counting only the lower endpoint. O(n) per query, O(1) extra space.

Yellow = current edge being tested. Red dashed = ray. Green/Red point result = inside/outside. Purple edge = ray crosses it (count++).

Time
O(n) per query
Space
O(1)
Method
