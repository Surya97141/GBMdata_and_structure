Convex Hull

Find the smallest convex polygon enclosing a set of points — the rubber-band boundary of a cloud of nails.

Imagine stretching a rubber band around a set of nails hammered into a board. When released, it snaps to the outermost nails — that shape is the convex hull. Every interior point is "hidden" behind some combination of hull vertices, and no part of the boundary curves inward.

Graham Scan sorts points by polar angle around the lowest point, then sweeps through them maintaining a stack where every turn is a left (counter-clockwise) turn — popping points that create a right turn. Jarvis March (gift wrapping) repeatedly picks the most counter-clockwise point from the current position, like wrapping a gift with tape.

Real-world uses include collision detection in games, geographic boundary computation, image processing (finding the bounding shape of a region), and robot path planning where you need the outermost safe boundary.

O(n log n)
Graham Scan time
O(nh)
Jarvis March (h = hull size)
O(n)
