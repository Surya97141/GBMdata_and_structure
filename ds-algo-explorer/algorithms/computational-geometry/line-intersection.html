Line Segment Intersection

Determine whether two line segments cross using the cross-product orientation test.

Two line segments AB and CD intersect if and only if A and B are on opposite sides of line CD, and C and D are on opposite sides of line AB. We test "which side" using the cross product (signed area of the triangle formed by three points).

The cross product d = (B−A) × (C−A) is positive if C is to the left of AB, negative if to the right, and zero if collinear. By computing four such values we can determine segment intersection without ever solving a system of equations — purely through sign checks.

This is the building block for sweep-line algorithms, polygon clipping (used in every graphics card), map overlay in GIS systems, and collision detection in physics engines where you check if a moving object's trajectory crosses a wall segment.
