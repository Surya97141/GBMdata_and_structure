Line Segment Intersection

CCW test using cross products — determine if two segments intersect in O(1).

Two segments AB and CD intersect iff A and B are on opposite sides of line CD AND C and D are on opposite sides of line AB. Side test uses cross product: cross(CD,CA) × cross(CD,CB) < 0. Handle collinear cases (cross = 0) by checking if endpoint lies on the segment. Total: O(1) per pair.

Steps show each cross product computation. Green dot = intersection point. Red segments = non-intersecting. Steps test each pair of pre-defined segments.
