Fibonacci Search

Use Fibonacci numbers to divide a sorted array into unequal segments.

Fibonacci Search uses Fibonacci numbers to divide a sorted array. It maintains three Fibonacci numbers (fibM2, fibM1, fibM) where fibM = fibM1 + fibM2. The probe index is offset + fibM2. Unlike Binary Search (which always picks the midpoint), this uses Fibonacci-based splits.

It achieves O(log n) like Binary Search but avoids division — useful on older hardware where division was expensive. Each step eliminates either the first fibM2 elements or the last fibM1 elements.
