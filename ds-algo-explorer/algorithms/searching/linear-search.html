Linear Search

Scan every element in sequence — no sorting required.

Linear Search checks each element from left to right until the target is found or the array is exhausted. It requires no pre-sorting and works on any array or linked list.

While O(n) seems slow, it is optimal for unsorted data (you can't do better without sorting first). For small arrays or infrequent searches, the constant-factor simplicity beats Binary Search's O(log n) when setup cost is considered.
