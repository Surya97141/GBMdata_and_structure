Interpolation Search

Estimate where the target should be — like searching a phone book.

Instead of always probing the midpoint (Binary Search), Interpolation Search estimates the probe position using linear interpolation: pos = lo + (target−a[lo]) / (a[hi]−a[lo]) × (hi−lo). On uniformly distributed data this converges in O(log log n).

Like looking up a name in a phone book: "Smith" starts near the back — you don't open it at the middle. It degrades to O(n) on non-uniform data (e.g. exponentially distributed values).
