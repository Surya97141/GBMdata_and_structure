Graph Coloring

Color vertices with k colors so no two adjacent vertices share a color.

Graph k-coloring: assign one of k colors to each vertex so that no two adjacent vertices have the same color. Backtracking: for each vertex (in order), try each color — if it doesn't conflict with neighbors, assign it and recurse. If no color works, backtrack.

Colored node = assigned color. Gray = uncolored. Crossed node (red) = conflict/backtrack. By the four-color theorem, any planar graph is 4-colorable.
