Permutation Generation

Generate all permutations of a set via recursive backtracking.

Generate all n! permutations of a string/array using backtracking: swap element at current position with each remaining element, recurse with position+1, then swap back (backtrack). This explores the recursion tree level by level.

The current partial permutation is shown in large text. Found permutations accumulate below. The swap-and-recurse pattern visits every leaf of the recursion tree, which corresponds to one complete permutation.
