Permutation Generation

Generate all n! arrangements of an array by systematically swapping elements — the classic swap-based backtracking approach.

A permutation is an ordered arrangement of all elements of a set. For n distinct elements there are n! (n factorial) permutations: 3 elements give 6, 4 give 24, 5 give 120. Generating them all is fundamental to solving problems like the Traveling Salesman Problem by brute force.

The swap-based backtracking algorithm works by fixing one element at each position. For position i, it swaps every element from index i to n-1 into position i, recurses to fill positions i+1 through n-1, then swaps back to restore the array. The "swap back" is the backtracking step.

Think of arranging books on a shelf: pick any book for the first slot, arrange the rest any way you like, then swap the first book back so you can try the next one. The animated swaps make each decision visible.
