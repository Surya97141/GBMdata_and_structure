Word Search

Find a word in a letter grid by backtracking in all 4 directions.

Word Search: given a 2D grid of letters and a target word, find if the word exists by navigating adjacent cells (up/down/left/right). Backtracking: mark cell as visited, try all 4 directions for the next letter, unmark (backtrack) if failed. No cell can be reused in the same path.

Yellow = current cell being explored. Purple = path so far (matching letters). Red = backtrack. Green = complete word found.

Time
O(N * 4^L)
Directions
4 (UDLR)
Revisit
Not allowed
Strategy
DFS backtrack
