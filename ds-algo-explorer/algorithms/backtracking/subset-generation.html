Subset Generation

Generate all 2ⁿ subsets of a set by recursively including or excluding each element — the power set via backtracking.

The power set of a set with n elements contains exactly 2ⁿ subsets (including the empty set and the full set). Backtracking generates them naturally by making a binary decision at each element: include it in the current subset, or exclude it.

Visualize a binary tree: at each level you branch left (include) or right (exclude). The leaf nodes at depth n each represent a unique subset. This is sometimes called a "decision tree" or "include-exclude" recursion, and it underpins the Subset Sum problem and the Knapsack problem.

Enter comma-separated values (e.g., 1,2,3,4) and press "Generate All Subsets" to watch every subset appear. The include/exclude decision tree is shown on the left; the found subsets appear as chips on the right.
