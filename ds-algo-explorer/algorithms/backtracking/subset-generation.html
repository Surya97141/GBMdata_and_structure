Subset Generation (Power Set)

Generate all 2^n subsets via include/exclude backtracking.

Power Set: generate all 2^n subsets of a set. Backtracking: for each element at position i, make two recursive calls — one including it in the current subset and one excluding it. The recursion tree has 2^n leaves, each a distinct subset.

Green = included in current subset. Yellow = currently deciding. Red = excluded. Each subset is recorded when all elements are decided (leaf of recursion tree).

2^n
Time
O(2^n)
