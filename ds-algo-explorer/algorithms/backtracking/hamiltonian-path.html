Hamiltonian Path

Find a path through a graph that visits every vertex exactly once — a classic NP-complete challenge solved with backtracking.

A Hamiltonian path is a path in a graph that visits each vertex exactly once. Unlike an Eulerian path (which traverses every edge), the Hamiltonian path focuses on vertices. Deciding whether such a path exists is NP-complete — no polynomial-time algorithm is known.

The backtracking approach builds the path one vertex at a time. From the current vertex, it tries each unvisited neighbor. If a dead end is reached (no unvisited neighbors but not all vertices visited), it backtracks — removing the last vertex from the path and trying a different neighbor. This is like a knight's tour problem on a chessboard.
