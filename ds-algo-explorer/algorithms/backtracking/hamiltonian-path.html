Hamiltonian Path

Find a path that visits every vertex exactly once.

Hamiltonian Path: find a path in a graph that visits every vertex exactly once. Backtracking: start from vertex 0, try adding adjacent unvisited vertices, recurse, backtrack if stuck. If path length equals N, a Hamiltonian path exists.

Green edges = current path. Orange node = current vertex. Gray nodes = unvisited. Blue nodes = visited in path. Red = backtracking. NP-complete — no known polynomial algorithm.
