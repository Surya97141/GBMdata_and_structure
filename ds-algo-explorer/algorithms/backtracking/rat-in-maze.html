Rat in a Maze

Backtracking path-finding from start to exit through a maze.

Rat in a Maze: given a grid where 1=open and 0=wall, find a path from (0,0) to (N-1,N-1) moving right/down. Backtracking: mark current cell as path, recurse right and down, backtrack (unmark) if dead end. All solutions can be enumerated by continuing after first success.

Orange = rat's current position. Purple = path so far. Gray = wall. Red = backtrack (dead end removed). Green = final solution path.

Time
O(2^(n2))
Moves
Right, Down
Grid
N x N
Strategy
DFS backtrack
