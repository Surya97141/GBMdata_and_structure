Sudoku Solver

Fill a 9×9 grid with digits 1–9 so each row, column, and 3×3 box contains all nine digits exactly once.

Sudoku is a constraint-satisfaction puzzle that backtracking handles elegantly. Starting from the first empty cell, the algorithm tries each digit 1–9 in order, checks whether placing it violates any row, column, or 3×3 box constraint, and recurses to the next empty cell on success.

If no digit fits in a cell, the algorithm backtracks to the previous empty cell and tries the next digit — exactly like an expert human solver who pencils in a digit, continues, and erases if a contradiction appears later. The cleverness is in the pruning: invalid digits are skipped immediately rather than explored.

A well-formed Sudoku puzzle has exactly one solution. The backtracking algorithm will always find it (or report that the puzzle is unsolvable). You can also edit the grey empty cells before solving.

Time Complexity
O(9^m)
m = number of empty cells
Space Complexity
O(m)
Recursion depth = empty cells
Valid Puzzles
~6.7×10²¹
