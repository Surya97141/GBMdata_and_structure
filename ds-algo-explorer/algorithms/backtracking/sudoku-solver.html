Sudoku Solver

Constraint-propagation backtracking to solve 9x9 Sudoku.

Sudoku Solver uses recursive backtracking: find an empty cell, try digits 1-9 checking row/column/box constraints, recurse if valid, backtrack (undo) if no digit works. The grid is solved when no empty cells remain.

Yellow cell = currently being tried. Green cells = successfully placed. Red cell = backtracking (removing a digit). Given clues are shown with gray background.
