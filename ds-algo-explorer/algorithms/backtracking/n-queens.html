N-Queens Problem

Place N queens on an NxN board so none attack each other.

N-Queens: place N queens on an NxN chessboard such that no two queens share the same row, column, or diagonal. Backtracking: place one queen per row (rows 0..N-1), try each column c, check if it conflicts with previously placed queens, recurse to next row if safe, backtrack if no column works.

Gold crown = placed queen. Red highlight = backtrack (removing queen). Attack lines are shown when conflict is detected.
