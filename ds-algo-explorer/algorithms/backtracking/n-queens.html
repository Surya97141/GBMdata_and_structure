N-Queens Problem

Place N queens on an N×N chessboard so no two queens threaten each other — row, column, or diagonal.

The N-Queens problem asks you to arrange N queens on an N×N chessboard such that no two queens share the same row, column, or diagonal. It is one of the most iconic backtracking problems because the search space is enormous yet pruning makes it tractable.

Think of it like placing chess pieces one row at a time. You try a column, check if any previously placed queen attacks the position, and move to the next row. If no column works in a row you backtrack to the previous row and shift that queen right. The analogy is solving a puzzle by erasing and retrying rather than starting fresh every time.

For N=8 there are 92 solutions. For N=4 there are 2. For N=1 there is 1. For N=2 and N=3 there are 0 solutions — a fact the solver discovers by exhausting all possibilities.
