N-Queens Problem

Place N queens on an N×N board — no two queens attack each other.

The N-Queens problem is one of the most iconic backtracking challenges: place N queens on an N×N chessboard so no two share a row, column, or diagonal. The search space is enormous (N! placements) but systematic pruning makes it tractable.

Algorithm: try each column in the current row, check conflicts against all previously placed queens, recurse to the next row. If no column works, backtrack (remove the last queen and advance its column). The animation shows every placement and backtrack step-by-step.
