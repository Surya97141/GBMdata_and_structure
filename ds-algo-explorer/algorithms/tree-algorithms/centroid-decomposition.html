Centroid Decomposition

Recursively split tree at centroids — O(n log n) for distance/path problems.

The centroid of a tree (or subtree) is a node whose removal leaves no component with more than n/2 nodes. After removing the centroid and recursing on each resulting subtree, any path in the tree passes through at most O(log n) centroids. This enables divide-and-conquer on paths: process all paths through the current centroid, then recurse.

Orange = current centroid being found. Green = centroid confirmed and removed. The recursion log shows the decomposition tree.
