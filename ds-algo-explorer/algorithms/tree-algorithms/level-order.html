Level Order Traversal (BFS)

Visit all nodes level by level using a queue — O(n) time and space.

Level order traversal visits nodes level by level, left to right. It uses a queue: start with root. For each node dequeued, visit it and enqueue its children. This gives the BFS tree structure. Used for finding shortest paths in unweighted trees, printing tree levels, and serialization.

Yellow = currently dequeued. Orange = in queue (waiting). Green = visited. Queue contents shown below the tree.
