Tree Traversals

Pre-order, In-order, Post-order, and Level-order traversals — O(n) each.

Tree traversals visit every node exactly once. Depth-first: pre-order (root, left, right), in-order (left, root, right — gives sorted order for BST), post-order (left, right, root — used for deletion/evaluation). All three are O(n) time and O(h) space (h = height).

Yellow = current node. Purple = already visited (in traversal order). The order sequence is shown below.
