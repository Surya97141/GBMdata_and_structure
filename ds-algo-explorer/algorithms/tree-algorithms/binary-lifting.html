Binary Lifting

Precompute 2^k-th ancestors for O(log n) LCA queries — sparse table on trees.

Binary lifting precomputes up[v][k] = 2^k-th ancestor of v. Built by: up[v][0] = parent[v], up[v][k] = up[up[v][k-1]][k-1]. For LCA queries: first equalize depths using powers of 2, then binary search for the highest bit where ancestors diverge. Each query is O(log n) after O(n log n) preprocessing.
