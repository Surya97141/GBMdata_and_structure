Heavy-Light Decomposition

Decompose tree into heavy paths — O(log²n) path queries with segment tree.

Each node designates its child with the largest subtree as its "heavy child". The heavy edges form disjoint paths (heavy chains) covering the entire tree. Any root-to-leaf path intersects at most O(log n) chains. Combined with a segment tree on the chain linearization, path queries become O(log²n).

Bold purple edges = heavy edges. Light edges = thin. The chains are listed. A node is heavy child if it has the largest subtree size among siblings.
