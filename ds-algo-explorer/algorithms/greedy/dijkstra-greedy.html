Greedy

Dijkstra's Algorithm

Find shortest paths greedily — always settle the nearest unsettled node first.

Dijkstra's algorithm finds the shortest path from a source node to every other node in a weighted graph (non-negative weights). It is fundamentally greedy: at each step, pick the unvisited node with the smallest current tentative distance and "settle" it.

The greedy choice is safe because edge weights are non-negative — if a node has the minimum tentative distance among unsettled nodes, no future path through other nodes can improve it. This exchange-argument reasoning gives us an O((V+E) log V) algorithm with a priority queue.

Dijkstra is the backbone of GPS routing, network routing protocols (OSPF), and any problem reducible to shortest paths on non-negative weighted graphs.
