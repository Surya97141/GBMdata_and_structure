Dijkstra's Shortest Path

Single-source shortest paths — greedy: always extract minimum-distance unvisited node.

Dijkstra's algorithm maintains a distance array d[]. Initially d[src]=0, all others ∞. Repeatedly extract the unvisited node u with minimum d[u]. Relax all edges from u: if d[u]+w(u,v) < d[v], update d[v]. Greedy choice works because all edges have non-negative weights.

Yellow = currently processing. Green = finalized (visited). Orange = relaxed neighbor. Distances shown above each node.
