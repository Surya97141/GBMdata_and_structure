Fractional Knapsack

Fill a knapsack to maximum value — fractions allowed, greedy by value/weight ratio.

You have a knapsack with a weight limit and a set of items, each with a weight and value. Unlike the 0/1 knapsack (where you must take or leave an item whole), here you can take any fraction of an item — like a thief scooping gold dust.

The greedy strategy is exact for this variant: compute the value-per-unit-weight ratio for every item, sort by ratio descending, then greedily take as much of each item as the remaining capacity allows.

This works because each unit of weight should carry the highest possible value, so we fill with the most "profitable" material first. The algorithm runs in O(n log n) and achieves the true optimum.
