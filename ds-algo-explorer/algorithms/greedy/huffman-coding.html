Huffman Coding

Optimal prefix-free code — greedily merge lowest-frequency nodes.

Huffman coding builds an optimal prefix-free binary code. Use a min-heap of character frequencies. Repeatedly remove the two nodes with lowest frequency, combine them into a new node with combined frequency, and insert back. The resulting trie gives optimal (minimum average length) codes. High-frequency characters get shorter codes.

The tree is visualized as it is built. Green = leaf (character). Purple = internal node. Numbers = frequencies.
