Greedy

Prim's Algorithm

Grow a Minimum Spanning Tree one cheapest crossing edge at a time.

Given a connected, weighted, undirected graph, the Minimum Spanning Tree (MST) is the subset of edges that connects all vertices with minimum total weight and no cycles. Prim's algorithm builds the MST greedily, starting from any node.

At each step, Prim looks at all edges that cross the "cut" — one endpoint in the current MST, one outside — and picks the cheapest crossing edge. This greedy choice is provably optimal by the Cut Property: the minimum-weight edge crossing any cut must be in some MST.

Prim's is conceptually similar to Dijkstra but tracks edge weight instead of path distance. It runs in O(E log V) with a priority queue and is well-suited for dense graphs.
