Rod Cutting Problem

Find the optimal cuts to maximize revenue from a rod.

Given a rod of length n and price table price[1..n] where price[i] is the value of a piece of length i, find the maximum revenue from cutting the rod. This is unbounded knapsack: dp[l] = max revenue for rod of length l. For each length l, try all first cuts i: dp[l] = max(price[i] + dp[l-i]). Build dp[0..n].

The visualization shows the DP array growing. Each bar's height shows the maximum revenue for that rod length. The traceback shows which cuts produce the optimal solution.

Time
O(n²)
Space
O(n)
