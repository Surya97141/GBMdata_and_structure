Catalan Numbers (DP)

Counting balanced parentheses, BSTs, triangulations — Catalan DP.

The n-th Catalan number C(n) counts: ways to parenthesize n+1 factors, BSTs with n nodes, triangulations of (n+2)-gon, paths that don't cross the diagonal, and many more. DP: C(0)=1, C(1)=1. C(n) = sum over i from 0 to n-1 of C(i)*C(n-1-i). Alternatively C(n) = C(2n,n)/(n+1).

The recurrence splits at each possible "root": left subtree has i nodes, right has n-1-i nodes. The simulation shows the DP array filling and for each new value, which pairs C(i)*C(j) contribute.
