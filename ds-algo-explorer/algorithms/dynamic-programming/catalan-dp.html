Catalan Numbers

A sequence that secretly governs parentheses, trees, and triangulations.

The Catalan numbers are a sequence of natural numbers that appear in many combinatorial problems: the number of valid bracket sequences with n pairs, the number of distinct Binary Search Trees with n nodes, the number of ways to triangulate a polygon, and many more.

The recurrence is elegant: C(0) = 1 and C(n) = Σ C(i) · C(n-1-i) for i from 0 to n-1. It mirrors the structure of dividing problems at a "root": everything to the left is one subproblem, everything to the right is another. DP lets us compute this bottom-up in O(n²).

The first few values are 1, 1, 2, 5, 14, 42, 132 — they grow exponentially, yet each is built from exactly those that came before.
