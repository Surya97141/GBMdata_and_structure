Coin Change (Min Coins)

Minimum number of coins to make amount — classic unbounded DP.

Coin Change: given coin denominations and a target amount, find the minimum number of coins to make that amount (unlimited coins of each denomination). DP recurrence: dp[a] = min over all coins c of (dp[a-c] + 1) where a ≥ c. Initialize dp[0]=0, dp[1..amount]=∞.

This is the dual of unbounded knapsack (minimize coins instead of maximize value). The dp array fills left to right. Red cells are impossible amounts. Green shows the final result at the target amount.

Time
O(amount × coins)
Space
O(amount)
Coins
Unlimited
Goal
Minimize count
