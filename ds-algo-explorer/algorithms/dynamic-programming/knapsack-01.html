0/1 Knapsack Problem

Maximize value while staying within weight capacity — each item used at most once.

The 0/1 Knapsack problem: given N items (each with weight and value) and capacity W, find the subset with maximum total value that fits in the knapsack. Each item is either taken (1) or not (0). The DP recurrence: dp[i][w] = max(dp[i-1][w], dp[i-1][w-wt[i]] + val[i]). Build an (N+1)×(W+1) table.

The table fills row by row. At each cell dp[i][w], we decide: skip item i (dp[i-1][w]) or take it (dp[i-1][w-wt[i]] + val[i]) if it fits. Trace back from dp[N][W] to find which items were chosen.

Time
O(N × W)
Space
O(N × W)
Items
Each used 0 or 1×
Optimal
Yes
