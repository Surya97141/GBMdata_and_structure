Unbounded Knapsack

Maximize value with unlimited copies of each item.

Unbounded Knapsack allows each item to be used unlimited times. Unlike 0/1 knapsack, we use a 1D DP array: dp[w] = maximum value achievable with capacity exactly w. Recurrence: dp[w] = max over all items i of (dp[w - wt[i]] + val[i]), when w ≥ wt[i]. The 1D array works because we can reuse items in the same capacity sweep.

This is also the basis for the Coin Change problem (minimize coins = same structure but minimize instead of maximize). The 1D approach uses O(W) space vs O(N×W) for the 2D table.
