Palindrome Partitioning

Cut a string into the fewest palindrome pieces possible.

Given a string, partition it so that every substring is a palindrome. We want the minimum number of cuts needed to achieve this. Every individual character is trivially a palindrome, so the answer is always at most n−1.

Two DP tables work together: first, isPalin[i][j] tells us in O(n²) time whether substring s[i..j] is a palindrome. Then, cut[i] stores the minimum cuts for s[0..i]. We set cut[i] = min over j where isPalin[j][i] of (cut[j-1] + 1).

The analogy: imagine breaking a word into "mirror pieces". You want as few breaks as possible, and a green cell in the palindrome table is your hint that a no-cut exists for that range.
