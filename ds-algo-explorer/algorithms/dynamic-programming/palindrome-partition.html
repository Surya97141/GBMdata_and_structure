Palindrome Partitioning II

Minimum cuts to partition a string into palindromes.

Palindrome Partitioning II: find the minimum number of cuts to partition string s into substrings where every part is a palindrome. Two-step DP: (1) isPal[i][j] = is s[i..j] a palindrome; (2) cuts[i] = minimum cuts for s[0..i]. cuts[i] = min over all j where isPal[j][i] of (cuts[j-1]+1). If isPal[0][i] is true, cuts[i]=0.

The palindrome table (green=palindrome, red=not) is filled first, then the 1D cuts array is derived. The answer is cuts[n-1].

Time
O(n2)
Space
O(n2)
Phases
isPal + cuts
Min cuts
0 if palindrome
