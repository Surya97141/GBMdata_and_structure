Word Break

Can a string be segmented into dictionary words? DP finds the answer.

Imagine running text through an autocorrect engine: "leetcode" has no spaces, but a smart parser knows it's "leet" + "code". Given a string and a dictionary, can the string be completely segmented into dictionary words?

We use a 1D boolean DP: dp[i] is true if s[0..i-1] can be segmented. Start with dp[0] = true (empty prefix). For each position i, we look back at every j: if dp[j] is true and s[j..i-1] is in the dictionary, then dp[i] = true.

The intuition: a green cell at position i is a valid "word boundary" — you've successfully assembled the prefix up to that point from dictionary words. We trace back through these boundaries to recover the actual word sequence.

O(n²·m)
