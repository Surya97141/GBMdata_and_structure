Word Break

Can the string be segmented into dictionary words?

Word Break: given a string s and dictionary of words, determine if s can be segmented into a sequence of dictionary words. 1D boolean DP: dp[i] = true if s[0..i-1] can be segmented. dp[0]=true (empty prefix). For each position i, try all j less than i: if dp[j] and s[j..i-1] is in dictionary, set dp[i]=true.

Classic string plus DP problem. Equivalent to reachability on a DAG where edges are dictionary words. Used in NLP, text segmentation (Chinese/Japanese), autocorrect.

Time
O(n2 x m)
Space
O(n)
Output
True / False
Uses
NLP, text seg
