Subset Sum

Can any subset of the given numbers sum to the target?

Subset Sum: given a set of non-negative integers, determine if any subset sums to a target value. 2D boolean DP: dp[i][s] = true if some subset of the first i numbers sums to s. Recurrence: dp[i][s] = dp[i-1][s] (skip num[i]) OR (s >= num[i] AND dp[i-1][s-num[i]]) (include num[i]).

Green cells (T) = achievable sum. Red cells (F) = not achievable. The bottom-right cell dp[N][target] gives the answer. Traceback reconstructs which elements were picked.

Time
O(N x T)
Space
O(N x T)
Output
True / False
Problem
NP-complete
