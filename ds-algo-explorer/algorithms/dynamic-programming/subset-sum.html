Subset Sum

Determine whether any subset of numbers adds up to a given target.

Given a set of positive integers and a target value, can you select a subset that sums to exactly that target? This is one of the most fundamental NP-complete problems, yet DP handles it efficiently when the target is not astronomically large.

We build a 2D boolean table: dp[i][s] is true if the first i items can form sum s. The transition is: we either skip item i (inherit dp[i-1][s]) or take it if s ≥ arr[i] and dp[i-1][s-arr[i]] is true.

Think of it like a cashier's coin problem: you want exact change. Each row of the table represents adding one more denomination. Green cells mean "reachable"; after filling the table we backtrace the highlighted path to recover the actual subset.
