Edit Distance (Levenshtein)

Minimum insert, delete, replace operations to transform one string to another.

Edit distance (Levenshtein distance) between two strings is the minimum number of single-character operations — insert, delete, or replace — needed to transform one string into the other. DP: dp[i][j] = edit distance between s1[0..i-1] and s2[0..j-1]. If s1[i-1]==s2[j-1]: dp[i][j]=dp[i-1][j-1]. Else: dp[i][j] = 1 + min(dp[i-1][j], dp[i][j-1], dp[i-1][j-1]) for delete/insert/replace.

Used in spell checkers, DNA sequence alignment, diff tools, and autocorrect. The DP table is filled row by row.

Time
O(m × n)
Space
O(m × n)
Ops
Insert, Delete, Replace
Applications
Spell check, DNA
