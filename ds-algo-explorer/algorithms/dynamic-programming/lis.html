Longest Increasing Subsequence

Find the longest strictly increasing subsequence using DP.

The LIS problem: given a sequence, find the longest subsequence (not necessarily contiguous) that is strictly increasing. DP approach: dp[i] = length of longest increasing subsequence ending at index i. For each i, look at all j < i where arr[j] < arr[i] and take dp[i] = max(dp[j]) + 1. Time O(n²).

The visualization shows bars representing array elements. When computing dp[i], we check all previous elements. Blue arrows indicate valid predecessor relationships. The LIS elements are highlighted green at the end.
