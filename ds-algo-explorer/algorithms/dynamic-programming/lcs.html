Longest Common Subsequence

Find the longest sequence of characters that appears in both strings in order.

LCS is the backbone of diff tools (git diff), DNA alignment, and plagiarism detectors. The DP recurrence: if X[i]==Y[j] then dp[i][j] = dp[i-1][j-1]+1; else dp[i][j] = max(dp[i-1][j], dp[i][j-1]).

Each cell fill is one step. After the table is complete, a traceback from dp[m][n] reconstructs the actual subsequence by following diagonal moves (character matches).
