Longest Common Subsequence

Longest subsequence common to both strings — a DP classic.

LCS: given two strings, find the longest sequence of characters that appears in both (not necessarily contiguous). DP: dp[i][j] = LCS length of s1[0..i-1] and s2[0..j-1]. If s1[i-1]==s2[j-1]: dp[i][j]=dp[i-1][j-1]+1. Else: dp[i][j]=max(dp[i-1][j], dp[i][j-1]).

Match cells (green) are the cornerstone: each extends the LCS by 1. Traceback: follow from dp[m][n] back to dp[0][0] — diagonal moves are matched characters. Used in diff utilities, file comparison, bioinformatics (DNA/RNA alignment).

Time
O(m x n)
Space
O(m x n)
Output
Length + sequence
Uses
diff, DNA align
Interactive Simulation
DP table (rows=s1, cols=s2).
