Longest Common Subsequence

Find the longest sequence of characters that appears in both strings in the same order.

LCS is the backbone of diff tools (like git diff), DNA sequence alignment, and plagiarism detectors. Given strings X and Y, a common subsequence is a sequence of characters that appears in both, in order, but not necessarily contiguously.

The DP recurrence: if X[i]==Y[j] then dp[i][j] = dp[i-1][j-1]+1; otherwise dp[i][j] = max(dp[i-1][j], dp[i][j-1]). After filling the table, backtrace from dp[m][n] to reconstruct the actual subsequence.

This runs in O(m·n) time and space. Real-world systems often use the space-optimised O(min(m,n)) variant, keeping only two rows at a time.
