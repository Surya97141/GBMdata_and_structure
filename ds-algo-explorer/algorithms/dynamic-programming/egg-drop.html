Egg Drop Problem

Find the critical floor using the fewest trials — even with limited eggs.

You have e eggs and a building with f floors. There is some critical floor F: eggs dropped from above F break; below or at F, they survive. You want to find F in the worst case using the fewest possible trials.

If you had infinite eggs, binary search (log₂ f trials) is optimal. But with limited eggs, breakage is costly. DP gives us: dp[e][k] = minimum trials needed with e eggs and k floors. We try every floor j: if it breaks we check below with e−1 eggs, if it survives we check above with e eggs — take the max (worst case), then minimize over all j.

The recurrence is dp[e][k] = 1 + min over j of max(dp[e-1][j-1], dp[e][k-j]). This classic puzzle builds intuition for adversarial worst-case reasoning combined with optimal substructure.
