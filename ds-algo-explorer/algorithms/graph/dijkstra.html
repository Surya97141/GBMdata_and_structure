Dijkstra's Algorithm

Greedy shortest path — always relax the closest unvisited node.

Dijkstra's finds the shortest path from a source node to all other nodes in a weighted graph with non-negative edges. It greedily picks the unvisited node with the smallest tentative distance, relaxes its edges, and marks it finalized. This greedy choice is safe because no future path can improve an already-extracted node.

Used in GPS routing, network routing protocols (OSPF), and as a subroutine in many graph algorithms. Requires non-negative weights — use Bellman-Ford for negative edges.
