Kruskal's Algorithm

Build MST by greedily adding the cheapest edge that doesn't create a cycle.

Kruskal's builds a Minimum Spanning Tree (MST) by sorting all edges by weight, then greedily adding the cheapest edge that doesn't form a cycle. Cycle detection uses a Union-Find (Disjoint Set Union) data structure: if both endpoints of an edge are already in the same component, adding it would form a cycle.

Excellent for sparse graphs. The Union-Find with path compression achieves near-O(1) per operation, making the overall algorithm O(E log E) dominated by sorting.
