Bellman-Ford Algorithm

Shortest paths with negative edges — relax every edge V-1 times.

Bellman-Ford computes shortest paths from a source by relaxing ALL edges V-1 times. A shortest path can have at most V-1 edges; so V-1 passes guarantee that every shortest path has been found. An additional pass detects negative cycles (if any distance improves, a negative cycle exists).

Slower than Dijkstra (O(VE) vs O(E log V)) but handles negative weights. Essential for routing protocols like RIP (Routing Information Protocol).
