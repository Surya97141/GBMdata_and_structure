Bridges & Articulation Points

Find critical edges and vertices whose removal disconnects the graph.

A bridge is an edge whose removal increases the number of connected components. An articulation point is a vertex whose removal disconnects the graph. Both are found in one DFS using disc[] (discovery time) and low[] (lowest disc reachable via subtree back-edges). Edge (u,v) is a bridge if low[v] > disc[u]. Node u is an articulation point if it has a child v with low[v] >= disc[u] (or it's the DFS root with 2+ children).

Critical for network reliability analysis: bridges are single points of failure in communication networks.

Time
O(V + E)
Bridges
low[v] > disc[u]
Art. Point
low[v] ≥ disc[u]
Passes
1 DFS
