Dinic's Algorithm

Max flow via level graph and blocking flows — much faster than Edmonds-Karp.

Dinic's algorithm achieves O(V²E) max flow. Each phase: (1) BFS builds a level graph (DAG of shortest-path layers). (2) DFS finds and saturates multiple paths simultaneously — a "blocking flow." Repeat until T is unreachable. One phase saturates a whole level, vs one path at a time for Edmonds-Karp.

Level graph nodes get BFS depth labels from the source. Only edges from level d to d+1 are used. After a blocking flow, shortest path distance must increase, giving at most V phases. Colors show which edges belong to the current level graph.
