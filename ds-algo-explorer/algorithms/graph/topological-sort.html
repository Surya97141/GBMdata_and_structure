Topological Sort (Kahn's)

Order nodes in a DAG so all edges point forward — Kahn's BFS approach.

Topological Sort linearly orders the vertices of a Directed Acyclic Graph (DAG) such that every directed edge u→v has u before v. Kahn's algorithm uses in-degrees: repeatedly remove nodes with in-degree 0 (no prerequisites), then update neighbours' in-degrees. If the result doesn't include all nodes, a cycle exists.

Applications: build systems (Makefile, Gradle), course scheduling, task ordering with dependencies, data pipeline orchestration.
