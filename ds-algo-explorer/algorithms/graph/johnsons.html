Johnson's Algorithm

All-pairs shortest paths for sparse graphs — smarter than Floyd-Warshall by reweighting edges.

Johnson's algorithm cleverly combines Bellman-Ford and Dijkstra to compute all-pairs shortest paths efficiently for sparse graphs. It reweights edges to eliminate negative weights, then runs Dijkstra from every vertex.

The trick: add a virtual source node q with zero-weight edges to all vertices. Run Bellman-Ford from q to get potentials h[v]. Reweight each edge (u,v,w) to w + h[u] - h[v] (always non-negative). Then Dijkstra runs safely from each vertex.

For sparse graphs, Johnson's O(VE + V² log V) beats Floyd-Warshall's O(V³). The reweighting does not change which path is shortest — it only ensures all weights are non-negative so Dijkstra can work.

O(VE + V²logV)
Space Complexity
O(V²)
