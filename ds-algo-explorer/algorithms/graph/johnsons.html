Johnson's Algorithm

All-pairs shortest paths — Bellman-Ford reweighting + Dijkstra for each source.

Johnson's algorithm finds shortest paths between all pairs of vertices in O(V²log V + VE). Step 1: Add a virtual node q with zero-weight edges to every vertex. Run Bellman-Ford from q to get potentials h[v]. Step 2: Reweight edges: w'(u,v) = w(u,v) + h[u] - h[v] ≥ 0. Step 3: Run Dijkstra from every node on the reweighted graph. Step 4: Recover actual distances: d(u,v) = dijkstra_dist + h[v] - h[u].

More efficient than Floyd-Warshall on sparse graphs. Works on graphs with negative edges (but not negative cycles). Red edges in the demo have negative weights.
