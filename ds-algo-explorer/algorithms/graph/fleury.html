Fleury's Algorithm

Find Eulerian path/circuit by avoiding bridges at each step.

Fleury's algorithm finds an Eulerian path (visits every edge exactly once). At each step, move along any available edge — but prefer non-bridge edges (edges whose removal won't disconnect the graph). A bridge is only taken if there's no other choice. An Eulerian circuit exists if all vertices have even degree. A path exists if exactly 2 vertices have odd degree (start from one, end at the other).

Fleury's is simple but slow O(E²) because it checks for bridges at every step using DFS. Hierholzer's algorithm solves the same problem in O(E).
