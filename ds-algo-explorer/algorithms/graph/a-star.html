A* Search Algorithm

Heuristic-guided shortest-path finding — faster than Dijkstra when you know the direction.

A* (pronounced "A-star") is a best-first search algorithm that finds the shortest path from a start node to a goal node. It combines the guarantee of Dijkstra's algorithm (optimal path) with the speed of greedy best-first search by using a heuristic function h(n) that estimates how close node n is to the goal.

The key insight is the evaluation function f(n) = g(n) + h(n), where g(n) is the actual cost from start to n, and h(n) is the heuristic estimate from n to goal. By always expanding the node with the lowest f value, A* efficiently navigates toward the goal. Think of it like GPS navigation: you know your current distance traveled (g) and estimate remaining distance (h) to pick the best route.

A* is admissible (finds optimal path) when the heuristic never overestimates. Manhattan distance works perfectly for grid navigation where diagonal moves aren't allowed — it always underestimates or exactly matches the true distance.
