Depth-First Search

Explore a graph as deep as possible before backtracking.

DFS dives as deep as possible along each branch before backtracking. It uses a stack (explicit or implicit via recursion): push the source, then repeatedly pop a node, mark it visited, and push its unvisited neighbours. DFS produces a DFS tree and classifies edges as tree, back, forward, or cross edges.

Core uses: cycle detection, topological sort, connected components, strongly connected components (Tarjan's, Kosaraju's), pathfinding in mazes, and solving puzzles like Sudoku.
