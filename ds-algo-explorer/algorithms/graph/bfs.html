Breadth-First Search

Explore a graph level by level — find the shortest path in an unweighted world.

BFS is like dropping a stone in water: ripples spread outward layer by layer. Starting from a source node, it visits all neighbors first, then their neighbors, ensuring every node is discovered at the minimum number of hops from the source.

Real-world analogy: social network "degrees of separation." BFS from your profile finds direct friends (level 1), friends-of-friends (level 2), and so on. It guarantees that when a node is first dequeued, its distance from the source is optimal.

The algorithm uses a queue (FIFO). Enqueue the source, mark it visited, then repeatedly dequeue a node, process it, and enqueue any unvisited neighbors. It terminates when the queue is empty.

Time Complexity
O(V + E)
Space Complexity
O(V)
Requires
