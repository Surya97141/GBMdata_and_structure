Breadth-First Search

Explore a graph level by level — shortest path in an unweighted world.

BFS explores a graph outward from a source, visiting all nodes at distance 1, then distance 2, and so on. It uses a queue (FIFO): enqueue the source, then repeatedly dequeue a node, process it, and enqueue its unvisited neighbours. When a node is first dequeued, its distance from the source is guaranteed optimal.

Real-world uses: shortest path in maps, social network degrees-of-separation, web crawling, and network broadcasting.
