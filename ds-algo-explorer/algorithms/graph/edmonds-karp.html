Edmonds-Karp Algorithm

BFS-based max-flow — always augment along the shortest path in the residual graph.

Edmonds-Karp is a specific implementation of Ford-Fulkerson that uses BFS to find augmenting paths instead of DFS. This seemingly small change provides a crucial guarantee: BFS always finds the shortest augmenting path (fewest edges), which ensures the algorithm terminates in O(VE²) time regardless of edge capacity values.

The key insight is that using BFS makes the augmenting path lengths non-decreasing across iterations. Since path length is at most V−1 and each edge can become "critical" (bottleneck) at most V/2 times before the path length increases, the total number of augmentations is bounded by O(VE).

Think of water flowing through pipes: BFS finds the most direct route each time you pump water. The pink highlighted path shows the current augmenting path found by BFS, and you can see flows on edges update as water fills the network.
