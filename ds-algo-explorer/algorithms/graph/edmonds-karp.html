Edmonds-Karp Max Flow

Ford-Fulkerson with BFS — finds shortest augmenting path each time.

Edmonds-Karp is Ford-Fulkerson with BFS instead of DFS for finding augmenting paths. BFS guarantees the shortest (fewest edges) augmenting path, giving O(VE²) complexity — much better than the O(E·maxFlow) of basic DFS, which can be slow on large capacities.

The key insight: BFS shortest paths guarantee that augmenting path lengths never decrease, so the total number of augmentations is bounded by O(VE). Each BFS augments the shortest path, systematically saturating edges level by level.
