Kosaraju's Algorithm (SCC)

Two-pass DFS reveals strongly connected components — the backbone of compiler analysis.

Kosaraju's algorithm finds all Strongly Connected Components (SCCs) in a directed graph in O(V+E) time. A SCC is a maximal set of vertices such that every vertex is reachable from every other vertex in the set.

The algorithm runs two DFS passes. First pass on the original graph: run DFS and push vertices onto a stack in order of decreasing finish time. Second pass on the transposed graph: pop vertices from the stack and run DFS — each DFS tree in the second pass is an SCC. The key insight: finishing order in the first DFS corresponds to topological order of the condensed SCC graph.

Applications include finding cycles in dependency graphs, analyzing social network clusters, deadlock detection in operating systems, and solving 2-SAT problems. Each SCC can be "contracted" into a single node, revealing the DAG structure of the original graph.
