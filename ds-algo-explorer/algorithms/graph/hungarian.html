Hungarian Algorithm

Optimal assignment — minimum cost perfect matching in bipartite graphs.

The Hungarian algorithm solves the assignment problem: given N workers and N jobs with costs, assign each worker to exactly one job to minimize total cost. Steps: (1) Subtract row/column minima. (2) Find minimum zero cover (König's theorem). (3) If cover = N, read off matching. Else adjust matrix. Repeat until cover size = N.

This runs in O(N³). The algorithm works on a cost matrix, reducing it until a zero-cost perfect matching exists. Zeros in the matrix represent "free" assignments that can be used.
