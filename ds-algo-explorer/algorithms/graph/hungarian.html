Hungarian Algorithm

Optimal assignment in O(n³) — minimize total cost by reducing a matrix of zeros.

The Hungarian Algorithm (Kuhn-Munkres) solves the assignment problem: given n workers and n jobs with a cost matrix, find the minimum-cost assignment of one worker to each job. It runs in O(n³) time and is provably optimal.

The algorithm works by transforming the cost matrix into a form that reveals a zero-cost assignment. Step 1: subtract each row's minimum. Step 2: subtract each column's minimum. Now find zeros that form an independent set (assignment). If fewer than n zeros can be found, cover all zeros with minimum lines, find the minimum uncovered value, subtract it from uncovered cells and add to doubly-covered cells, then repeat.

Used everywhere from task scheduling and logistics to machine learning (tracking objects between frames). The name honors Harold Kuhn who published it in 1955 — though it was originally developed by Jenő Egerváry, a Hungarian mathematician.
