Borůvka's Algorithm

Build MST in rounds — each component grabs its cheapest outgoing edge.

Borůvka's algorithm builds an MST in O(log V) rounds. Each round: every connected component finds its minimum-weight outgoing edge and adds it to the MST (using Union-Find to merge components). Components halve each round, giving O(E log V) total time.

The oldest MST algorithm (1926). More naturally parallelizable than Kruskal or Prim — different components can find their cheapest edge simultaneously, making it efficient in parallel/distributed systems.
