Borůvka's Algorithm

Parallel minimum spanning tree construction — pick the cheapest escape from every island simultaneously.

Borůvka's algorithm, invented in 1926 by Otakar Borůvka, was the first MST algorithm ever published. It works by treating each node as its own component initially, then in each phase finding the minimum-weight edge leaving each component and merging connected components together.

Unlike Prim's (one node at a time) or Kruskal's (one global edge sort), Borůvka works in parallel phases. In each phase, every component simultaneously finds its cheapest outgoing edge. This makes at least half the components merge each round, giving at most O(log V) phases. Think of isolated villages each building the cheapest road to their nearest neighbor — they eventually form a connected region.

Borůvka's algorithm is the foundation of efficient parallel MST algorithms and is used in filter-Kruskal and randomized MST algorithms. Its time complexity is O(E log V) and it works beautifully on dense graphs.
