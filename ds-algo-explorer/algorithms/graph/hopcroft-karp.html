Hopcroft-Karp Algorithm

Maximum bipartite matching using BFS layers + DFS augmenting paths.

Hopcroft-Karp finds a maximum matching in a bipartite graph in O(E√V) time. Each phase: (1) BFS from all unmatched left nodes to find shortest augmenting paths (by layers). (2) DFS to find a maximal set of vertex-disjoint augmenting paths of the BFS length. Each phase increases the matching size by multiple paths at once.

Augmenting path alternates between unmatched and matched edges. After O(√V) phases, the matching is maximum. Left nodes are shown in purple; right nodes in green. Matched edges are bold.
