Hopcroft-Karp Algorithm

Maximum bipartite matching in O(E√V) — BFS layered graph + simultaneous DFS augmentations.

Hopcroft-Karp finds maximum matching in a bipartite graph in O(E√V) time. Like Dinic's algorithm for max-flow, it works in phases: each phase runs a BFS to build a layered graph of alternating paths, then finds multiple augmenting paths simultaneously using DFS.

The algorithm maintains matchedL[] and matchedR[] arrays — the current matching. An augmenting path alternates between unmatched and matched edges, starting and ending at unmatched vertices. By finding all shortest augmenting paths in one BFS phase, the algorithm achieves its superior complexity.

Bipartite matching has countless real-world applications: assigning workers to jobs, students to schools, organs to recipients, tasks to processors. Hopcroft-Karp is the algorithm of choice when you need the most efficient maximum matching.
