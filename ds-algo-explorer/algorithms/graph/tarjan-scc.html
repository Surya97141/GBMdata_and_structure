Tarjan's SCC Algorithm

Single-pass DFS with a stack — disc[] and low[] values reveal SCCs on the fly.

Tarjan's algorithm finds all Strongly Connected Components (SCCs) in a directed graph using a single DFS pass in O(V+E) time. It's more efficient than Kosaraju's (which needs two passes) and reveals SCCs as they're completed during the DFS.

Each node gets a discovery time disc[u] and a "low link" low[u] — the lowest discovery time reachable from the subtree rooted at u (including back edges). A node u is an SCC root if low[u] = disc[u], meaning no node in its subtree can reach an ancestor of u. When an SCC root is found, pop all nodes above it from the stack — they form the SCC.

Tarjan's algorithm outputs SCCs in reverse topological order, which is extremely useful. It's used in 2-SAT problem solving, finding feedback vertex sets, and computing the condensation graph of a directed graph for simplified analysis.
