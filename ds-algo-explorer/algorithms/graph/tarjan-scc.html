Tarjan's SCC Algorithm

Find all Strongly Connected Components in one DFS pass using low-link values.

Tarjan's algorithm finds all Strongly Connected Components (SCCs) — maximal subsets where every node is reachable from every other. It runs one DFS, maintaining discovery times (disc[v]) and low-link values (low[v] = earliest disc reachable from v's subtree). When disc[v] == low[v], v is the root of an SCC — pop the stack to find all members.

Single O(V+E) pass is more efficient than Kosaraju's two-DFS approach. Used for condensation DAGs, circuit layout, compiler dependency analysis.
