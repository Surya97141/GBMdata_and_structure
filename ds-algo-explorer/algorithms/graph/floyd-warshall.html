Floyd-Warshall Algorithm

All-pairs shortest paths via dynamic programming on intermediate vertices.

Floyd-Warshall finds shortest paths between every pair of nodes using DP. For each intermediate node k, it checks if routing through k shortens the path from i to j: dist[i][j] = min(dist[i][j], dist[i][k] + dist[k][j]). After considering all k, the matrix contains all-pairs shortest paths.

O(V³) time — slower than running Dijkstra from each vertex (O(VE log V)) for sparse graphs, but simpler and handles negative weights. Also detects negative cycles (dist[i][i] < 0).
