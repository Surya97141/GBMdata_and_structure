Floyd-Warshall Algorithm

All-pairs shortest paths in one elegant dynamic programming sweep through intermediate nodes.

Floyd-Warshall computes the shortest path between every pair of vertices. The key insight: for each intermediate node k, check whether routing through k improves the path from i to j. Repeat for all k from 0 to V-1.

Real-world analogy: a flight connection matrix. For each hub airport k, we check every pair (i, j): is it cheaper to fly i→k→j than the direct route? After considering all hubs, we have the globally optimal connection for every pair.

The algorithm is beautifully simple: three nested loops. The outermost loop picks the intermediate node k. D[i][j] = min(D[i][j], D[i][k] + D[k][j]). Watch the matrix update as k increases.

Time Complexity
O(V³)
Space Complexity
O(V²)
