Hierholzer's Algorithm

Eulerian circuit via stack-based DFS — O(E) time, much faster than Fleury's.

Hierholzer's finds an Eulerian circuit in O(E) time. The algorithm uses a stack: push nodes as you explore unused edges. When you reach a dead end (no unused edges), pop from the stack to the circuit. Keep exploring from nodes with unused edges. The circuit is built by interleaving these sub-cycles.

The key advantage over Fleury's: no bridge detection needed. The stack naturally handles dead ends — you just backtrack and splice in the current path. This makes it O(E) vs Fleury's O(E²).
