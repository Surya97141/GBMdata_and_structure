Hierholzer's Algorithm

Efficient Euler circuit via sub-circuit merging — O(E) and far faster than Fleury's.

Hierholzer's algorithm finds an Eulerian circuit in O(E) time — dramatically faster than Fleury's O(E²). The key insight: instead of checking for bridges at every step, build sub-circuits and merge them together. Start a DFS from any vertex and follow edges until you return to the start, forming a sub-circuit.

Then find any vertex in the current circuit that still has unused edges, start a new sub-circuit from there, and splice it into the original circuit. Repeat until all edges are used. The merging step is what makes it efficient: no bridge detection needed.

Hierholzer's algorithm is used in DNA fragment assembly (finding Eulerian paths in de Bruijn graphs), Chinese Postman problem solutions, and routing algorithms for garbage collection trucks needing to traverse every street.
