Ford-Fulkerson Max Flow

Find maximum flow via DFS augmenting paths on the residual graph.

Ford-Fulkerson finds maximum flow from source to sink by repeatedly finding an augmenting path (using DFS) in the residual graph and pushing flow along it. Each augmentation increases flow by the bottleneck capacity. The algorithm terminates when no augmenting path exists — at that point, by the max-flow min-cut theorem, max flow equals min cut.

Edges show flow/capacity. The residual graph has forward edges (remaining capacity) and backward edges (current flow, allowing "undoing" of flow). This demo shows each DFS path and the flow update step by step.
