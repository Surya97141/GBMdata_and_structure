Ford-Fulkerson Algorithm

The foundational max-flow method — find augmenting paths in the residual graph until none remain.

Ford-Fulkerson is the foundational algorithm for computing maximum network flow. The idea: repeatedly find a path from source S to sink T with available capacity (an "augmenting path"), push as much flow as possible along it (the bottleneck capacity), and update the residual graph. Repeat until no augmenting path exists.

The residual graph is crucial: for every forward edge (u,v) with flow f and capacity c, there's a backward edge (v,u) with capacity f. This allows the algorithm to "undo" bad flow decisions by sending flow backwards. Using DFS to find paths, the algorithm may not always terminate for irrational capacities, but works for integer capacities.

Toggle the "Residual" view to see back-edges. The pink highlighted path shows the current DFS augmenting path. Ford-Fulkerson proves the Max-Flow Min-Cut theorem: maximum flow equals minimum cut capacity.
