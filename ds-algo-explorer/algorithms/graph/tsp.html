Traveling Salesman Problem

Find shortest Hamiltonian cycle — bitmask DP gives optimal O(2ⁿ·n²) solution.

TSP asks: given N cities and distances, find the shortest tour visiting every city exactly once and returning to start. The bitmask DP approach uses state dp[mask][i] = minimum cost to visit exactly the cities in mask, ending at city i. Fill in 2ⁿ × n states. Time: O(2ⁿ·n²), Space: O(2ⁿ·n). Optimal but only feasible for small N.

This demo shows the bitmask DP table filling for N=5 cities, then traces back the optimal tour. The nearest-neighbor heuristic comparison shows a typical greedy approximation.
