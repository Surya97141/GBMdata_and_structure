Fast Power (Exponentiation by Squaring)

Compute b^n in O(log n) multiplications via recursive halving — visualized as a recursion tree.

Fast power (also called "exponentiation by squaring") halves the exponent at each step: pow(b, n) = pow(b, n/2)² if n is even, or b × pow(b, n-1) if n is odd. This reduces O(n) multiplications to O(log n).

The recursion tree is like a binary tree where each node computes one squaring. The depth is log₂(n), so there are at most 2 log₂(n) multiplications total. For n=1 billion, that's only 60 multiplications instead of 1 billion.

Unlike modular exponentiation, fast power computes exact values — useful for polynomial exponentiation, matrix exponentiation (Fibonacci in O(log n)), and computing binomial coefficients with exact arithmetic.
