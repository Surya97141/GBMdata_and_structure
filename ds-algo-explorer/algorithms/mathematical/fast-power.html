Fast Power (Binary Exponentiation)

Compute x^n in O(log n) multiplications — recursive and iterative.

pow(x,n) = pow(x²,n/2) if n is even; x × pow(x²,(n-1)/2) if n is odd. Each recursive call halves n, giving O(log n) depth. The iterative version processes binary digits of n: square always, multiply by x when bit=1.

Each bit of the exponent is shown. Purple = bit contributes to result. Yellow = currently processing bit.
