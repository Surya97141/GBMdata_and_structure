Miller-Rabin Primality Test

Probabilistic prime test — O(k log²n) with error probability 4^(-k).

Write n-1 = 2^s × d. For each witness a, compute x = a^d mod n. Repeatedly square: if x becomes -1 (mod n), n passes for this witness. If x reaches 1 without first being -1, n is composite. k random witnesses give error probability 4^(-k).

Each witness round is shown. Green = passes this witness. Red = composite detected. Yellow = currently testing.
