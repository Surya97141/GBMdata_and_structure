Chinese Remainder Theorem

Find x such that x ≡ rᵢ (mod mᵢ) for pairwise coprime moduli.

CRT: given x ≡ r₁ (mod m₁), x ≡ r₂ (mod m₂), … with pairwise coprime mᵢ, there is a unique solution mod M = m₁×m₂×…. For each i: Mᵢ = M/mᵢ, then find yᵢ = Mᵢ⁻¹ mod mᵢ (using extended GCD), then x = Σ rᵢ×Mᵢ×yᵢ mod M.

Each congruence step is shown. Final answer satisfies all congruences simultaneously.

Conditions
pairwise coprime
Solution
Unique mod M
Used in
RSA, NTT, scheduling
Time
O(k log M)
