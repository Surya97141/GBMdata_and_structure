Sieve of Sundaram

Find odd primes by marking composite numbers i+j+2ij, then converting remainders via 2k+1.

The Sieve of Sundaram (1934, S.P. Sundaram) takes a different approach: instead of directly marking composites, it marks numbers of the form i + j + 2ij where 1 ≤ i ≤ j. The key insight is that 2k+1 is composite exactly when k = i+j+2ij for some valid i,j — because 2(i+j+2ij)+1 = (2i+1)(2j+1).

So: create a grid of numbers 1 to (N-2)/2. Mark every number of the form i+j+2ij. The unmarked k values correspond to odd primes 2k+1. Then add 2 separately. Result: all primes up to N.

The sieve is pedagogically interesting as it proves that finding odd primes reduces to a simple arithmetic marking on a smaller grid. The grid has only N/2 elements rather than N, giving a slight memory advantage over the standard sieve.
