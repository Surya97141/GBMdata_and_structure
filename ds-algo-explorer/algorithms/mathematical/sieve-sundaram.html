Sieve of Sundaram

Find all odd primes up to 2N+2 by marking i+j+2ij ≤ N.

The Sieve of Sundaram marks numbers of the form i+j+2ij (i,j ≥ 1). If k is not marked, then 2k+1 is prime. Start with {1,2,…,N}, eliminate all i+j+2ij, then 2×remaining+1 gives all odd primes up to 2N+1. Finally add 2 as the only even prime.

Yellow = current (i,j) being eliminated. Purple = marked composite. Green = surviving primes mapped to 2k+1.

Time
O(n log n)
Key formula
i+j+2ij
Output
2k+1 primes
Invented
1934 by Sundaram
