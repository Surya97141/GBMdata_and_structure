Catalan Numbers

C(n) = Σ C(i)×C(n-1-i) — counts BSTs, bracket sequences, polygon triangulations.

Catalan numbers: C(0)=1, C(1)=1, C(n) = C(0)×C(n-1) + C(1)×C(n-2) + … + C(n-1)×C(0). They count many combinatorial structures: binary search trees on n keys, valid parenthesization, monotonic lattice paths, polygon triangulations.

Bar chart shows C(0)…C(n). Yellow = current bar being computed. Purple dashed lines = contributing C(i)×C(n-1-i) pairs. Green = finalized.

Formula
C(2n,n)/(n+1)
