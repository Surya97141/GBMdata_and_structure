Extended Euclidean Algorithm

Find GCD and Bézout coefficients x, y such that ax + by = gcd(a,b).

The Extended Euclidean Algorithm not only computes gcd(a,b) but also finds integers x and y such that ax + by = gcd(a,b) (Bézout's identity). This is essential for finding modular inverses: if gcd(a,m)=1, then x is a's inverse mod m.
