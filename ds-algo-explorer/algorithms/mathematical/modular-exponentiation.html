Modular Exponentiation

Compute base^exp mod m in O(log exp) — the core of RSA and Diffie-Hellman.

Fast modular exponentiation uses repeated squaring: if the exponent bit is 0, just square; if 1, square and multiply by base. Taking mod at each step keeps numbers small. This replaces O(exp) multiplications with O(log exp).

The table shows each bit of the exponent, the running square, and the accumulated result (multiplied in only when bit=1). Yellow = current step, purple = multiplied into result.
