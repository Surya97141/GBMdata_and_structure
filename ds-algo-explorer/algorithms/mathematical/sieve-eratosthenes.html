Sieve of Eratosthenes

Find all primes up to N in O(n log log n).

Start with all numbers marked prime. For each prime p starting from 2, mark all multiples of p (from p²) as composite. Numbers not crossed out are prime. The key insight: we start from p² because all smaller multiples were already marked by smaller primes.

Yellow = current prime p being sieved. Purple = multiples being crossed out. Green = confirmed prime. Gray = composite.
