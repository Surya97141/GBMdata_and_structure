Manacher's Algorithm

Find all palindromic substrings in O(n) time using a clever center-expansion mirror trick.

Manacher's algorithm is like using a mirror. If you know a large palindrome centered at C, and you're looking at position i inside it, then its mirror i' (= 2C - i) already tells you how far the palindrome at i can extend — you don't need to re-expand from scratch.

The key insight: transform the string by inserting '#' between every character (and at ends). This converts all palindromes to odd-length, simplifying the algorithm. We maintain a center C and right boundary R of the rightmost known palindrome. For each new position, we either use the mirror value directly or expand further.

The amortized cost is O(n) because R only moves right, never left. Applications include DNA palindrome detection, syntax highlighting, and competitive programming problems asking for longest palindrome.
