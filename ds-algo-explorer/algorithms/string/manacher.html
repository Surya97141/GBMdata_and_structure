Manacher's Algorithm

All palindromic substrings in O(n) — longest palindrome found.

Manacher's algorithm finds the longest palindromic substring in O(n) by inserting separators (#) to unify odd/even palindromes, then computing P[i] = palindrome radius at position i. It reuses previously computed radii using a center C and right boundary R, similar to Z-algorithm's Z-box trick.

P[i] (shown above each cell) = palindrome radius. Purple = within current right boundary. Yellow = currently computing. Green = longest palindrome found so far.
