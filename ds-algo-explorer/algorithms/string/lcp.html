LCP Array (Kasai's Algorithm)

Longest Common Prefix between consecutive suffix array entries in O(n).

The LCP array stores LCP[i] = length of the longest common prefix between SA[i] and SA[i-1] (consecutive suffixes in sorted order). Kasai's algorithm builds it in O(n) using the rank array (inverse of SA): if suffix at position i has LCP k, then the suffix at position i+1 has LCP at least k-1 (the key insight).

The table shows: rank (SA position), suffix text, and LCP value. LCP is built by iterating over original positions using the rank array.
