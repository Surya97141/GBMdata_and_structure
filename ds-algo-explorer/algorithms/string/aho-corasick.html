Aho-Corasick

Multi-pattern string search in O(n + total matches).

Aho-Corasick builds a trie from all patterns, then computes failure (suffix) links similar to KMP's LPS. Searching the text follows trie edges, falling back via failure links on mismatch. All patterns are found in a single O(n) pass — ideal for searching many patterns simultaneously.

Used in network intrusion detection (Snort), antivirus scanning, and grep. Current trie node is shown. Matches are highlighted green in the text.
