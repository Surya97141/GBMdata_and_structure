Suffix Array Construction

Sort all suffixes lexicographically — foundation of advanced string algorithms.

A suffix array is the sorted order of all suffixes of a string. This simulation uses the naive O(n² log n) comparison-sort approach for clarity, building intuition before the O(n log n) or O(n) SA-IS algorithm. The sorted suffix array enables O(log n) string search with binary search.

Each sort step is shown as a comparison between two suffixes. The final sorted table is the suffix array SA[] — the rank of each suffix determines its position in sorted order.
