KMP — Knuth-Morris-Pratt

Linear-time pattern matching using the failure function.

KMP avoids redundant character comparisons by precomputing the LPS (Longest Proper Prefix which is also Suffix) array for the pattern. When a mismatch occurs at pattern index j, instead of shifting by 1, KMP jumps to lps[j-1] — resuming from the longest useful prefix already matched.

Phase 1 builds the LPS array in O(m). Phase 2 searches in O(n). Total O(n+m) — never re-examines a text character.
