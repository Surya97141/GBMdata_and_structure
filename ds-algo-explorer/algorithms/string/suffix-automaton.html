Suffix Automaton (SAM)

Minimal DFA accepting all suffixes — built online in O(n).

A Suffix Automaton (SAM) is the smallest DFA that accepts all suffixes of a string. It is built online: extend character by character, creating at most 2 new states per character. Each state represents an equivalence class of substrings. The SAM has at most 2n-1 states and 3n-4 transitions.

The simulation shows the state table growing as each character is added. Each state has transitions (edges), a link (suffix link), and len (longest suffix length in this class).
