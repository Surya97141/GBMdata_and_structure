Suffix Automaton (SAM)

The smallest DFA that recognizes all substrings — built incrementally in O(n) time with suffix links.

A Suffix Automaton (SAM) is the most compact structure for representing all substrings of a string. Unlike a suffix trie (which has O(n²) nodes), the SAM has at most 2n-1 states. Think of it as a subway map where every station represents a set of substrings, and you can reach every substring by following directed edges.

Each state has a len (longest substring it represents), a link (suffix link to the state representing proper suffix classes), and transitions (labeled edges to next states). The algorithm adds one character at a time, creating at most 2 new states per character.

Applications include counting distinct substrings in O(n), finding the longest common substring of two strings, and string matching — all fundamental in competitive programming and bioinformatics.
