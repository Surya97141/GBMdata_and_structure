Z-Algorithm

Z-array: longest prefix of the string starting at each position.

The Z-algorithm computes Z[i] = length of the longest substring starting at position i that is also a prefix of the full string. For pattern matching, create S = pattern + "$" + text; positions where Z[i] == m indicate a match. The algorithm maintains a Z-box [L,R] to avoid redundant comparisons.

Z-box (purple) = current known matching range. Yellow = position being computed. Values above each cell = Z-value. Green = match found in combined string.

Time
O(n + m)
Space
O(n + m)
Z[0]
Undefined (len)
Match if
Z[i] == m
