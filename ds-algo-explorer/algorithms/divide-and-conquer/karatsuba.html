Karatsuba Multiplication

Multiply two n-digit numbers using 3 multiplications instead of 4 — O(n^1.585).

Split x = a·10^m + b and y = c·10^m + d. Naive: ac, ad, bc, bd (4 multiplications). Karatsuba computes: z2=ac, z0=bd, z1=(a+b)(c+d)-z2-z0 (only 1 extra mult). Result: z2·10^2m + z1·10^m + z0. Uses 3 multiplications instead of 4.

Purple = sub-multiplications. Yellow = current step. Green = final result.

Time
O(n^1.585)
Multiplications

vs Long mult
