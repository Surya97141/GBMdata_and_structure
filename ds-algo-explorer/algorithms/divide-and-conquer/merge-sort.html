Merge Sort

Divide array in half, sort each half, merge — O(n log n) guaranteed.

Merge sort recursively splits the array into halves until single elements, then merges sorted halves by comparing elements. It is stable, O(n log n) in all cases, and the basis for external sorting (tapes, large files).

Bars represent array values. Yellow = elements being compared during merge. Green = placed in correct position. Purple = left/right subarrays in current merge.
