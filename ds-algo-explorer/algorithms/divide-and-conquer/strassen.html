Strassen Matrix Multiplication

Multiply 2×2 matrices using 7 multiplications instead of 8 — O(n^2.807).

Strassen's algorithm computes 2×2 matrix product using 7 multiplications (vs standard 8). Recursively applied, this gives O(n^log₂7) ≈ O(n^2.807). The 7 products M1..M7 are computed from sums/differences of submatrices, then combined additively to get C.

Each of the 7 Strassen products is shown with its formula. The final matrix C is built by combining them.
