Binary Search

Search sorted array in O(log n) by halving search space each step.

Binary search maintains a [lo, hi] range. Each iteration computes mid = (lo+hi)/2 and compares a[mid] to target. If equal, found. If target > a[mid], search right half. Otherwise left half. Each step eliminates half the elements.

Yellow = current mid element being compared. Purple = active search range. Gray = eliminated range. Green = found.

Time
O(log n)
Space
O(1)
Requires
Sorted array
Steps
⌊log₂ n⌋ + 1
