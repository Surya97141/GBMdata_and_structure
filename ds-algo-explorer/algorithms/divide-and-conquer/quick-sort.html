Quick Sort

Choose pivot, partition around it, recurse — O(n log n) average, O(n²) worst.

Quick sort selects a pivot, partitions the array so all smaller values are left and larger are right, then recurses on both halves. In-place and cache-friendly. Worst case O(n²) on sorted input — avoided by random pivot selection.

Orange = pivot. Yellow = elements being compared to pivot. Purple = in left partition (≤ pivot). Green = fully sorted.
