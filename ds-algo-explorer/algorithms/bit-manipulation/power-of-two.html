Power of Two Check

n & (n-1) == 0 iff n is a power of two — O(1) using Brian Kernighan's trick.

A power of two has exactly one set bit: 1, 10, 100, 1000... Subtracting 1 flips that bit and sets all lower bits: 01, 011, 0111... AND-ing gives 0 iff n was a power of two. Combined with n>0 check: `n > 0 && (n & (n-1)) == 0`. This is O(1) — no loops, no counting.

Purple = set bits in n. Yellow = set bits in n-1. Green = AND result = 0 (power of two). Steps demonstrate the check for a batch of numbers.
