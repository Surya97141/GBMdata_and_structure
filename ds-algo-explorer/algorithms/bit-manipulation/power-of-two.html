Power of Two

O(1) power-of-2 checks, next-power-of-2, and highest-set-bit using bitwise tricks.

A number is a power of 2 if and only if it has exactly one set bit in its binary representation — like 1 (0001), 2 (0010), 4 (0100), 8 (1000). The trick n & (n−1) clears the lowest set bit. If the result is zero, there was only one set bit: it's a power of 2.

Finding the next power of 2 ≥ n uses a sequence of OR-and-shift operations to "smear" the highest set bit downward, then adds 1. This is useful for sizing hash tables and buffers to the nearest power of 2.

The highest set bit (floor of log₂) tells you the bit-length of a number — useful in prefix-free codes, segment trees, and binary lifting.
