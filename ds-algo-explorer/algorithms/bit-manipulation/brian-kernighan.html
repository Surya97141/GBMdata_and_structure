Bit Manipulation

Brian Kernighan's Bit Count

Count set bits in O(k) where k = number of set bits — faster than O(log n) naive.

The trick: n & (n-1) clears the lowest set bit. Repeat until n=0, counting iterations. Number of iterations = popcount(n). Subtracting 1 from n flips the lowest set bit and all trailing zeros, so ANDing with n clears exactly one set bit per iteration. Total iterations = number of 1-bits.

Purple = 1-bits. Yellow = bit being cleared this iteration. Green = cleared (0). Each row shows n, n-1, n & (n-1).
