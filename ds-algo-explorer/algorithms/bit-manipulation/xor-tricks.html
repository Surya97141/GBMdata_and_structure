XOR Tricks

XOR's self-inverse property powers swap-without-temp, missing number, and duplicate detection — all in O(n) time O(1) space.

XOR (exclusive-or) has a magical property: a ^ a = 0 and a ^ 0 = a. This means if you XOR all elements in an array where every element appears twice except one, the duplicates cancel out, leaving only the unique element — no extra memory needed.

Swapping two numbers without a temporary variable — a^=b; b^=a; a^=b — is a classic interview trick. Each step transforms (a,b) until they exchange values using only XOR operations on the original storage locations.

Finding the missing number in 1..n follows the same principle: XOR all array elements with all numbers 1..n. Pairs cancel, the lone survivor is the missing number.
