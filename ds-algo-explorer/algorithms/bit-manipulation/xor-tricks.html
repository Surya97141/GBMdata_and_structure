XOR Tricks

Find the single non-duplicate element in O(n) time, O(1) space using XOR.

XOR key identities: x⊕x=0, x⊕0=x, commutative and associative. If every element appears twice except one, XOR-ing all elements cancels all pairs, leaving the unique element. No extra space needed. Also used for: swap without temp (a⊕=b; b⊕=a; a⊕=b), detect two non-duplicates, bit position tricks.

Yellow = currently XOR-ing. Purple = paired/cancelled. Green = final unique element found.
