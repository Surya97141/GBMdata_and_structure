Bitmask DP — Travelling Salesman

TSP in O(n² × 2ⁿ) using bitmask to track visited cities — exact solution for small n.

Bitmask DP represents the set of visited cities as a single integer. dp[mask][v] = minimum cost to visit exactly the cities in "mask", ending at city v. Transition: for each city u not in mask, dp[mask|(1<<u)][u] = min(dp[mask][v] + cost[v][u]). Final answer = min(dp[all][v] + cost[v][0]) for all v.

Yellow = currently computing. Purple = already computed. The mask column shows which cities have been visited as a bitmask.
