Merge Sort

Divide, sort each half, then merge — guaranteed O(n log n) every time.

Merge Sort splits the array into halves recursively until each subarray has one element (trivially sorted), then merges adjacent sorted subarrays back together. The merge step compares the front elements of two sorted runs and takes the smaller one.

Unlike quicksort, performance is guaranteed O(n log n) regardless of input. The simulation shows the bottom-up iterative version, merging runs of size 1, 2, 4, 8… until the full array is sorted.
