Comb Sort

Bubble Sort with a shrinking gap — eliminates turtles fast.

Comb Sort improves Bubble Sort by comparing elements with a large gap first (shrink factor ≈ 1.3), eliminating "turtles" (small elements near the end that drag down Bubble Sort). The gap shrinks each pass until it reaches 1, at which point it degenerates to Bubble Sort — but by then the array is nearly sorted.

In practice it runs in O(n log n) for most inputs, and is significantly faster than Bubble Sort while being just as simple to implement.
