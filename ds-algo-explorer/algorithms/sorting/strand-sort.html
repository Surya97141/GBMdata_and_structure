Strand Sort

Pull sorted strands from the input and merge them one by one.

Strand Sort repeatedly scans the remaining input and extracts the longest increasing "strand" (subsequence) starting from the first element. Each extracted strand is merged with the running output list. Repeat until the input is empty.

It is adaptive: O(n) on already-sorted data (one strand pulls everything), O(n²) worst case (reverse-sorted). It naturally exploits existing sorted order in the input.
