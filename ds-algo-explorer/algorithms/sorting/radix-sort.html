Radix Sort

Sort digit by digit from least significant to most — no comparisons.

Radix Sort (LSD variant) applies a stable sort on each digit position, starting from the least significant digit. Because each single-digit sort is stable, relative order from previous passes is preserved, building up a fully sorted result.

Time complexity is O(d·(n+10)) where d = number of digits. It beats comparison sorts when d is small relative to n. This simulation sorts by ones, tens, hundreds… using counting sort as the per-digit stable sort.
