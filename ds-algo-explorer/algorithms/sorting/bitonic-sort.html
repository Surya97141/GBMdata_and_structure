Bitonic Sort

Parallel-friendly sort using compare-and-swap networks — ideal for GPUs.

Bitonic Sort recursively creates "bitonic sequences" (first increasing then decreasing) and merges them into a fully sorted sequence. Every compare-and-swap at the same level can execute in parallel, making it perfect for GPUs and hardware sorters.

Input size must be a power of 2. This simulation pads the array with large sentinel values if needed. Sequential version is O(n log² n) comparisons — more than merge sort but every level is parallelisable.
