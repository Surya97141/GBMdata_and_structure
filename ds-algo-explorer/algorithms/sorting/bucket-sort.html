Bucket Sort

Distribute into range buckets, sort each, then concatenate.

Bucket Sort divides the value range into equal-width buckets, distributes each element into its bucket, sorts each bucket individually (using Insertion Sort here), then concatenates all buckets.

It achieves O(n) average time when input is uniformly distributed across the range. It is ideal for floating-point data in [0,1) or integers with known range. Worst case (all in one bucket) degrades to O(n²).
