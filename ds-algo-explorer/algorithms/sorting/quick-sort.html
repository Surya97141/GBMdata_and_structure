Quick Sort

Partition around a pivot and recurse — the fastest in practice.

Quick Sort picks a pivot, partitions the array so all elements smaller are left and all larger are right, then recursively sorts each partition. The pivot ends up in its final position after partitioning.

This simulation uses the last element as pivot and Lomuto partitioning. Average case is O(n log n); with random pivot selection, worst-case O(n²) is extremely rare in practice.
