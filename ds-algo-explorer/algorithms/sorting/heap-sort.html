Heap Sort

Build a max-heap, then extract the maximum one by one.

Heap Sort works in two phases: first build a max-heap from the unsorted array (heapify phase), then repeatedly extract the maximum (root) by swapping it to the end and sifting down the new root to restore the heap.

It achieves O(n log n) worst-case like Merge Sort, but sorts in-place like Quick Sort — the best of both worlds. The heap is visualised as a bar chart; bars at the right end get "locked" as sorted elements accumulate.
