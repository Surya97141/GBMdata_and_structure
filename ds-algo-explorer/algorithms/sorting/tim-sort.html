Tim Sort

Python's and Java's built-in sort — Insertion Sort on small runs, then Merge Sort.

Tim Sort is a hybrid stable sort that splits the array into small "runs" (size RUN, here 4), sorts each with Insertion Sort (which is fastest for small arrays), then merges adjacent sorted runs using Merge Sort until the whole array is sorted.

It is the default sort in Python, Java (Arrays.sort for objects), and many other languages. O(n) on already-sorted or reverse-sorted data; O(n log n) otherwise.
