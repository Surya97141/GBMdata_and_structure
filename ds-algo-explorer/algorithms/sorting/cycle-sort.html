Cycle Sort

Minimum writes to sorted position — ideal when write cost dominates.

Cycle Sort minimises the number of array writes by detecting permutation cycles and rotating each element directly to its final position. Each element is written at most once. Total writes = n − (number of cycles).

It runs in O(n²) time but makes O(n) writes — the theoretical minimum. Useful when writes to memory or disk are expensive (e.g. flash storage with limited erase cycles).
