Counting Sort

Count occurrences, prefix-sum, then reconstruct — no comparisons needed.

Counting Sort counts how many times each value appears, then transforms the count array into a prefix-sum (cumulative count) array, and finally reconstructs the output array by placing each element at its correct position in one right-to-left pass.

It runs in O(n + k) where k is the value range — faster than any comparison sort when k is small. Elements must be non-negative integers. Values are clamped to 0–99 in this simulation.
