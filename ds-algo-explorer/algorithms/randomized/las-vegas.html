Las Vegas Algorithm

Always correct, random runtime — illustrated with random 4-Queens placement.

A Las Vegas algorithm always produces a correct answer, but its running time is random. The N-Queens example demonstrates: randomly pick a safe column for each row. If no safe column exists for any row, restart entirely. Each trial is correct if it completes; otherwise retry. Expected restarts is small for N=4.

Orange = queen placed. Red = attacked cell. Yellow = trying. Green = all queens placed safely.
