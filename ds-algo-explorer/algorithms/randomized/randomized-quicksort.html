Randomized Quicksort

Random pivot selection eliminates worst-case O(n²) for sorted inputs — expected O(n log n).

Deterministic quicksort always picks the last (or first) element as pivot, making sorted arrays worst-case O(n²). Randomized quicksort picks a uniformly random pivot before each partition. Any input gives O(n log n) expected time because no adversary can construct a worst-case input.

Orange = pivot. Yellow = comparing element. Purple = left partition (≤ pivot). Red = right partition. Green = sorted in place.
