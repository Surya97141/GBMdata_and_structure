Quad Tree

Recursively subdivide 2D space into 4 quadrants — fast spatial queries on 2D point sets.

A quadtree recursively divides a 2D region into four quadrants: NW, NE, SW, SE. Each leaf holds at most one point (or a small bucket). Insertion: find which quadrant the point falls in, recurse until leaf; if leaf overflows, subdivide. Range search: if query region intersects quadrant, recurse into it — prune quadrants that don't intersect. Used in 2D collision detection, image compression, geographic data.

Each division line shows a quadrant split. Blue = current node boundary being examined. Orange = active split. Green = point found in range query. Purple = point outside query.
