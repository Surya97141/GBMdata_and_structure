R-Tree

Minimum bounding rectangles for spatial indexing — nearest neighbor, range queries on 2D objects.

An R-tree indexes spatial objects (rectangles, polygons, points) using minimum bounding rectangles (MBRs). Each internal node holds 2-M child MBRs; leaves hold actual objects. Insert: choose subtree with minimum MBR expansion → insert → split if overflow. Query: prune nodes whose MBR doesn't intersect query. Used in PostGIS, SQLite spatial extension, geographic databases for 2D range and nearest-neighbor queries.

Blue = level-1 MBRs (root). Purple = level-2 MBRs. Green = objects matching query. Orange = current node being checked. Yellow dashed = query rectangle.
