Rope

A binary tree for efficient string concatenation, splitting, and indexing at scale.

A Rope is a binary tree data structure designed for efficient manipulation of very long strings. Unlike a flat array-based string where inserting a character in the middle requires shifting O(n) bytes, a Rope can concatenate two strings in O(1) (just create a new root node) and split a string at any index in O(log n).

Internal nodes store the total length of all leaves in their left subtree (the "weight"), which enables O(log n) character indexing by walking down the tree. Leaf nodes hold the actual string fragments. Real-world text editors like VS Code's Monaco use a similar structure (called a "piece table") to handle large files efficiently.

Real-world analogy: Instead of one long paper scroll, imagine a filing system where documents are broken into labeled chunks stored in folders. To append a new section, you just add a new folder label — you do not re-copy everything.
