Dancing Links (DLX)

Knuth's Algorithm X with efficient cover/uncover via doubly-linked circular lists.

Dancing Links (DLX) is Knuth's implementation of Algorithm X for exact cover. Given a 0/1 matrix, find a set of rows where each column has exactly one 1. DLX uses doubly-linked circular lists — "covering" a column removes it and all conflicting rows in O(1); "uncovering" restores them (the links "dance"). Used to solve Sudoku, N-Queens, pentomino puzzles.

We solve exact cover on a small 4-column, 6-row matrix. Yellow = row selected, Green = column covered, Red = conflicting row removed. Algorithm backtracks when stuck.
