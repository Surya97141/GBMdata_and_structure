Dancing Links

Knuth's Algorithm X with circular doubly-linked lists for exact cover problems.

Dancing Links is Donald Knuth's efficient implementation of Algorithm X for solving exact cover problems. An exact cover asks: given a universe of columns and a set of rows, select a subset of rows such that every column is covered exactly once. This directly models Sudoku, N-Queens, and pentomino tilings.

The "dancing" refers to the doubly-linked list manipulation used to cover and uncover columns during the depth-first search. When a column is covered, its nodes are unlinked from their rows and vice versa — but the links are preserved, so uncovering (backtracking) restores them in O(1) per node without any memory allocation.

Real-world analogy: imagine assigning tasks to teams. Dancing Links finds which teams to select so that every task is assigned to exactly one team — no task missed, none assigned twice.
