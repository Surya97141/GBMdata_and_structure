Matrix (2D Array)

2D grid — O(1) cell access. Row/column traversal, transpose, multiplication.

A matrix is a 2D array of R rows × C columns. Cell access: O(1). Row-major memory layout means row traversal is cache-friendly; column traversal is not. Key operations: transpose (swap rows/cols), matrix multiply (O(n³) naive), rotate 90°. Used in graphics, DP tables, graph adjacency, image processing.

Yellow = cell currently being processed. Purple = cell already visited. Green = result written. Orange = pivot/diagonal element.
