Matrix Operations

2D arrays powering linear algebra, graphics, and machine learning.

A matrix is a rectangular grid of numbers arranged in rows and columns. It is the fundamental object of linear algebra — every transformation in computer graphics (rotation, scaling, projection) is encoded as a matrix multiplication. When you rotate a 3D game character or apply a filter in image processing, a matrix is doing the work under the hood.

Key operations include the transpose (flipping rows and columns), matrix multiplication (composing two transformations), and row echelon form (Gaussian elimination for solving linear systems). The determinant of a square matrix tells you whether it is invertible — a zero determinant means the system has no unique solution.

Real-world analogy: think of a matrix as a recipe card grid — rows are ingredients, columns are steps, and matrix multiplication is like following two recipes sequentially to produce a combined dish.
