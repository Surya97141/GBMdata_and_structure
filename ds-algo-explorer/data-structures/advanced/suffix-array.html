Suffix Array

All suffixes of a string, sorted — enabling fast pattern matching and LCP queries.

A Suffix Array is a sorted array of all suffixes of a given string. For the string "banana", its suffixes are "banana", "anana", "nana", "ana", "na", "a" — and the suffix array gives their sorted order by index: [5, 3, 1, 0, 4, 2] (i.e., "a", "ana", "anana", "banana", "na", "nana").

The companion LCP Array stores the length of the longest common prefix between consecutive suffixes in sorted order. Together they support pattern matching in O(m + log n) time and are the backbone of DNA sequence alignment tools like BWA.

Real-world analogy: think of a phonebook that lists every possible "tail" of every name, sorted alphabetically. Want to find all entries containing "ana"? Binary search the sorted list in milliseconds.
