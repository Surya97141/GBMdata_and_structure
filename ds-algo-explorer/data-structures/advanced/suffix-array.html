Suffix Array

Sorted array of all suffixes — O(n log n) build, O(m log n) pattern search.

A suffix array SA is an array of starting indices of all suffixes of a string, sorted lexicographically. Build: O(n log n) with prefix-doubling. Pattern search: binary search on SA in O(m log n) where m=|pattern|. Combined with LCP array enables O(n+m) search. Used in bioinformatics, full-text search, data compression.

Building: all n suffixes sorted. Search: binary search finds range [lo,hi] where suffixes start with pattern. Yellow = current comparison, Purple = sorted suffix, Green = match found.
