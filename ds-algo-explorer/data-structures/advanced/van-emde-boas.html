Van Emde Boas Tree

O(log log U) for all operations on integers in universe [0, U). Faster than O(log n).

Van Emde Boas (vEB) tree supports insert, delete, search, successor, predecessor on integer keys from universe {0..U-1} in O(log log U) time. Structure: universe split into √U clusters; each cluster is a vEB(√U). Summary vEB tracks which clusters are non-empty. Key insight: each operation makes at most one recursive call on a sub-problem of size √U, giving T(U) = T(√U) + O(1) → T(U) = O(log log U).
