Skip List

Layered linked lists — O(log n) expected search using express lanes. Simpler than balanced BSTs.

A skip list is a randomized data structure with multiple levels of linked lists. Level 0 contains all elements; higher levels skip over elements with probability 1/2 per level. Search: start high, go right until overshoot, drop down, repeat. Expected O(log n) search, insert, delete. Much simpler to implement than AVL or red-black trees. Used in Redis sorted sets.

Yellow = currently visiting node. Orange = forward pointer followed. Green = target found. Levels shown top-to-bottom (highest = fastest express lane).
