Cartesian Tree

BST by index, min-heap by value — in-order traversal recovers original array.

A Cartesian tree is a binary tree built from an array where: (1) in-order traversal gives the original array, and (2) each node's value is less than or equal to its children (min-heap property). Built in O(n) using a stack. Root = global minimum. Left subtree = subarray left of root, right subtree = subarray right of root. Used in RMQ, treap construction, and suffix array LCP queries.

Building: process array left-to-right, maintaining a stack of right-spine nodes. Orange = global minimum (root). Yellow = node being inserted. Purple = already placed in tree.
