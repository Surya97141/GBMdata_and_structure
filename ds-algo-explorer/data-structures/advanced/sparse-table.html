Sparse Table

O(1) range minimum query after O(n log n) preprocessing. Immutable arrays only.

A sparse table precomputes answers for all intervals of length 2^k. For RMQ: ST[i][k] = min of A[i..i+2^k-1]. Query [l,r]: find largest k s.t. 2^k ≤ r-l+1, then return min(ST[l][k], ST[r-2^k+1][k]). The two intervals overlap but min is idempotent — overlapping is fine. Build: O(n log n). Query: O(1). Cannot handle updates (immutable).

Yellow = cell being computed during build. Purple = cells used in final query. Green = query result.
