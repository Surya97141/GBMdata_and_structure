Octree

An octree recursively divides 3D space into 8 octants (cubes). Each internal node has up to 8 children. A node splits when it contains more than the bucket capacity. Point insertion: navigate to correct octant based on (x,y,z) coordinates relative to the node center. Range/radius queries prune octants that don't intersect the query region. Used in 3D graphics (frustum culling, ray tracing, LOD), LiDAR point cloud processing, 3D collision detection.
