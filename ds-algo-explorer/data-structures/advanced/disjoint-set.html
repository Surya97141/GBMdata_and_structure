Disjoint Set (Union-Find)

Track connected components — nearly O(1) amortized with path compression + union by rank.

A disjoint set (union-find) maintains a partition of elements into disjoint sets. Operations: find(x) returns the representative (root) of x's set; union(x,y) merges the sets. Path compression (flatten on find) and union by rank (attach smaller tree under larger) give amortized O(α(n)) per operation — practically constant. Used in Kruskal's MST, cycle detection, network connectivity.
