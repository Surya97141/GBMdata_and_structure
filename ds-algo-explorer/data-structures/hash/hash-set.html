Hash Set

Unordered collection of unique elements — O(1) avg add, contains, remove. No duplicates.

A hash set stores unique elements using a hash table (no values, just keys). Add/contains/remove are all O(1) average. Used when you need fast membership testing without duplicates. Supports set operations: union, intersection, difference. Python set, Java HashSet use this structure.

Yellow = bucket being examined. Green = element found. Steps show intersection of two sets: {10,20,30,40,50} ∩ {20,40,60}.
