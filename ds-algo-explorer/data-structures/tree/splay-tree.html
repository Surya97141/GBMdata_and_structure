Splay Tree

Self-adjusting BST — accessed node splayed to root. Amortized O(log n) per operation.

A splay tree is a self-adjusting BST. Every access (search, insert, delete) rotates ("splays") the accessed node to the root using zig, zig-zig, or zig-zag rotations. Recently accessed nodes are near the root, giving good cache behavior for skewed access patterns. Amortized O(log n) per operation via potential argument.

Yellow = node being splayed. Orange = rotation being applied. Green = node successfully at root. Steps show the splay path for a chosen node.
