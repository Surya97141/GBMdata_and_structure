Fibonacci Heap

Collection of min-heap-ordered trees — O(1) amortized insert and decrease-key. Theoretical optimum.

A Fibonacci heap is a collection of min-heap-ordered trees (roots linked in a doubly linked list). Insert and decrease-key are O(1) amortized; extract-min is O(log n) amortized. This gives Dijkstra O(E + V log V) total — optimal for dense graphs. Work is lazily deferred (consolidation happens only at extract-min). Named for Fibonacci numbers appearing in tree size bounds.

Yellow = minimum node (root of min-tree). Orange = node being inserted. Green = node after decrease-key. Steps show insert and extract-min on a 6-node Fibonacci heap.
