Trie

A prefix tree — lightning-fast autocomplete and prefix search.

A Trie stores strings character-by-character, with each node representing one character. Words sharing a common prefix share the same path from the root. A special end-of-word flag marks complete words.

When you type "Jo" in a contacts app, it instantly suggests "John", "Jose", "Jordan" because a trie lets you walk a prefix in O(m) steps — independent of how many words are stored.

Insert / Search / Delete
O(m) where m = word length
Space
O(ALPHABET × N)
