Heap (Min-Heap)

Complete binary tree with heap property — min at root. O(log n) insert, O(log n) extract-min.

A min-heap is a complete binary tree where each node ≤ its children. Stored as an array: node i has children at 2i+1, 2i+2 and parent at ⌊(i-1)/2⌋. Insert: append to end, sift up. Extract-min: swap root with last, sift down. Used in Dijkstra, Prim, priority queues, heapsort.

Yellow = nodes being compared/swapped during sift-up. Green = newly inserted node. Array below shows heap array representation.
