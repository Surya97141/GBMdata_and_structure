Binary Tree

Each node has at most two children — left and right. Foundation for BSTs, heaps, and expression trees.

A binary tree is a hierarchical data structure where each node has at most two children, called left and right. Height h gives O(h) traversal. A balanced tree has h=O(log n); a skewed tree degrades to O(n). Common traversals: pre-order (root→L→R), in-order (L→root→R), post-order (L→R→root), level-order (BFS).

Yellow = current node. Purple = visited. Green = done. Steps show in-order traversal visiting nodes left-to-right.
