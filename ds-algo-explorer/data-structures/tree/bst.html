Binary Search Tree (BST)

BST property: left subtree < node < right subtree. O(log n) average search, insert, delete.

A BST maintains the invariant: all values in the left subtree are less than the node, and all in the right subtree are greater. Search, insert, and delete are O(h) where h is height. For a balanced tree h=O(log n); degenerate (sorted input) gives h=O(n).

Yellow = current comparison node. Orange = go right (target > node). Purple = go left (target < node). Green = found/inserted.
