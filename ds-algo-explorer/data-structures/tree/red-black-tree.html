Red-Black Tree

Self-balancing BST with 5 color invariants — O(log n) guaranteed. Used in Java TreeMap, C++ std::map.

A red-black tree is a BST where each node is colored red or black, with 5 invariants ensuring no path is more than 2× longer than any other: (1) every node is red or black; (2) root is black; (3) leaves (NIL) are black; (4) red nodes have black children; (5) all paths from node to leaves have equal black-height. This guarantees h ≤ 2log(n+1).
