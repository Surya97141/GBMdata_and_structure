Fenwick Tree (BIT)

Compact prefix-sum structure — O(log n) query and update using bit manipulation tricks.

A Binary Indexed Tree (BIT) stores partial sums. BIT[i] stores the sum of arr[i-lowbit(i)+1..i] where lowbit(i)=i&(-i). Prefix sum query[1..r]: traverse by removing lowest set bit (r -= r&(-r)). Update[i]: propagate by adding lowest set bit (i += i&(-i)). Simpler and faster in practice than segment tree for prefix sums.

Yellow = currently accessed BIT cell. Purple = already accumulated. Each cell labeled with its index, value, and range it covers.
