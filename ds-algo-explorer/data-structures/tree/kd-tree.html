k-d Tree

Space-partitioning BST for k-dimensional points — nearest neighbor and range queries.

A k-d tree (k-dimensional tree) is a BST that recursively splits the k-dimensional space by alternating dimensions. At depth d, split on dimension d mod k. Building takes O(n log n). Nearest-neighbor search takes O(√n) average in 2D. Range queries take O(n^(1−1/k) + m) where m is the output size. Used in machine learning (k-NN), ray tracing, and GIS.

Yellow = current splitting node. Blue line = vertical split (x-axis). Red line = horizontal split (y-axis). Green = point in range. Steps show tree construction by alternating splits.
