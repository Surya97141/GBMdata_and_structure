KD-Tree

A space-partitioning tree that organizes 2D (or higher-dimensional) points for blazing-fast nearest-neighbor and range searches.

A KD-Tree (K-Dimensional Tree) is a binary tree that recursively partitions a k-dimensional space. At each level, a splitting plane divides points along one axis, alternating axes as you go deeper. In 2D: level 0 splits by X (vertical line), level 1 splits by Y (horizontal line), level 2 by X again, and so on.

Imagine organizing city locations on a map. First you split the map into "left" and "right" halves using a vertical line through the median longitude. Each half is then split into "top" and "bottom" halves by a horizontal median latitude. Repeat recursively. Now to find the nearest restaurant to your current location, you only need to search a tiny region of the tree — not every city.

Nearest-neighbor search prunes entire subtrees when the splitting plane is farther from the query point than the current best distance. In practice this eliminates most of the tree, giving sub-linear search times on typical inputs. KD-Trees power location services, collision detection in games, and k-NN machine learning classifiers.
