Suffix Tree

Compressed trie of all suffixes — O(n) build, O(m) pattern search. Used in bioinformatics.

A suffix tree for string S is a compressed trie containing all suffixes of S. Each edge is labeled with a substring (stored as index pointers). Build in O(n) with Ukkonen's algorithm. Pattern matching: traverse the tree reading the pattern — O(m) to find if pattern exists, where m = pattern length. Used in longest common substring, repeated substrings, bioinformatics (DNA search).
