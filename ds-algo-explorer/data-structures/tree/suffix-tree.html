Suffix Tree

A compressed trie of all suffixes of a string — the backbone of ultra-fast substring search and DNA analysis.

A Suffix Tree is a compressed trie that stores every possible suffix of a given string. For the word "banana", suffixes are: "banana", "anana", "nana", "ana", "na", "a" — all inserted into a single trie that shares common prefixes. Edges in the compressed form carry substrings rather than single characters, making the structure compact.

Imagine an index at the back of a book. Instead of listing every page a word appears on, a suffix tree answers "does this substring exist?" in O(m) time where m is only the query length — no matter how long the original text is. Bioinformatics uses suffix trees to search billions of DNA base pairs for genetic patterns in milliseconds.
