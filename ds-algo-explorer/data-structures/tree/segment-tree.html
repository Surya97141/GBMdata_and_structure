Segment Tree

Range queries and point updates in O(log n). Each node stores the result for a subarray.

A segment tree is a binary tree where each node stores an aggregate (sum, min, max) of a subarray. Leaves hold individual elements. Internal nodes combine their children. Query [l,r]: decompose into O(log n) disjoint nodes. Update: propagate changes up the path. Used in competitive programming for range sum/min/max.

Yellow = currently visiting. Green = node fully within query range (contributes to answer). Orange = partial overlap (recurse). Gray = outside range. Query shown step by step.
