Queue

FIFO — First In, First Out. Enqueue at rear, dequeue at front in O(1).

A queue is a FIFO structure: the first element enqueued is the first dequeued. Enqueue adds to the rear; dequeue removes from the front. Both O(1). Used for BFS, task scheduling, producer-consumer, print spoolers, and any first-come-first-served scenario.

Yellow = front (next to dequeue). Green = just enqueued. Red = being dequeued. Queue flows left (front) to right (rear).
