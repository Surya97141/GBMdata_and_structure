Array

Contiguous fixed-size collection with O(1) random access by index.

An array stores elements in contiguous memory. Access by index is O(1) because the address = base + i×size. Insertion/deletion at an arbitrary position requires shifting elements: O(n). Appending to the end is O(1) amortized (dynamic arrays). Searching is O(n) unsorted or O(log n) sorted.

Yellow = currently accessing. Green = inserted. Red = about to be deleted. Indices shown below each cell.
