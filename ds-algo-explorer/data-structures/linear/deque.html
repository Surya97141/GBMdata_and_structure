Deque

Double-ended queue — push/pop at both front and rear in O(1).

A deque (double-ended queue) supports O(1) insertion and deletion at both ends. It generalizes both stack (LIFO) and queue (FIFO). Used in sliding window maximum, palindrome checking, work-stealing schedulers, and BFS variants. Implemented with a circular buffer or doubly linked list.

Purple = front end. Green = rear end. Yellow = element being operated on. Red = element being removed.
