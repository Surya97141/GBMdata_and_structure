Stack

LIFO — Last In, First Out. Push and pop from top in O(1).

A stack is a LIFO structure: the most recently pushed element is the first to be popped. Operations: push (add to top), pop (remove from top), peek (view top). All are O(1). Used for function call stacks, undo operations, expression evaluation, DFS iterative, bracket matching.

Yellow = current top. Green = just pushed. Red = being popped. Stack grows upward.
