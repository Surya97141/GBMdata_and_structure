Linked List

Dynamic nodes with next pointers — O(1) insert/delete at head, O(n) access.

A linked list consists of nodes, each holding a value and a pointer to the next node. Unlike arrays, nodes are not contiguous in memory. Insertion/deletion at the head is O(1). Random access requires traversal from head: O(n). Used where frequent insertions/deletions at known positions outweigh access time.

Yellow = currently traversing. Green = newly inserted. Red = being deleted. Arrow = next pointer.
