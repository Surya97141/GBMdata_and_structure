Incidence Matrix

V×E matrix — row=vertex, col=edge. Entry=1 if vertex is endpoint of edge.

An incidence matrix is V×E where M[v][e]=1 if vertex v is an endpoint of edge e (0 otherwise). For directed graphs: +1 for tail (source), -1 for head (destination). Space is O(V×E). Rarely used for algorithms but useful for linear algebraic analysis of graph properties (cycle detection, cut spaces). Each column has exactly 2 ones.
