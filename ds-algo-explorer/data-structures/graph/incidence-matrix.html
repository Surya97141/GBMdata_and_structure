Incidence Matrix

A V×E matrix that records which vertices are endpoints of which edges — powerful for directed graphs and circuit analysis.

How it Works

The incidence matrix is V rows × E columns. Column j represents edge j. For an undirected edge between vertices i and k, both cells [i][j] and [k][j] = 1. For a directed edge from i to k: cell [i][j] = +1 (source) and cell [k][j] = −1 (destination). All other cells = 0.

Real use: electrical circuit analysis (Kirchhoff's laws), network flow, and linear algebra on graphs. The incidence matrix reveals structure that adjacency matrices cannot — like identifying self-loops (a +1 and −1 in the same row) or parallel edges as separate columns.
