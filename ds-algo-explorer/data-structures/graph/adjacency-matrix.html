Adjacency Matrix

V×V matrix where M[i][j] = edge weight — O(1) edge lookup, O(V²) space.

An adjacency matrix is a V×V 2D array where M[u][v] holds the edge weight (or 1/0 for unweighted). Edge existence check is O(1). Finding all neighbors of a vertex is O(V) — must scan the entire row. Space is always O(V²) even for sparse graphs. Best for dense graphs or when frequent edge-existence queries are needed.

Yellow = currently querying cell. Purple = edge exists (non-zero). Gray = no edge (0). Steps show row-by-row neighbor scan.
