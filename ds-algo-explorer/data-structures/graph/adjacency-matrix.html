Adjacency Matrix

A V×V grid where cell [i][j] = 1 if an edge exists from vertex i to vertex j — instant edge lookup, quadratic space.

How it Works

An adjacency matrix is a 2-D array of size V×V. If there is an edge from vertex i to vertex j, we set A[i][j] = 1. For undirected graphs the matrix is symmetric. Edge lookup is O(1) — just check one cell — but storing the matrix always costs O(V²) space, even for sparse graphs.

Think of a social network with a yes/no friendship grid: row = person A, column = person B, cell = are they friends? Great for dense graphs and algorithms like Floyd-Warshall; wasteful for sparse ones like road networks.
