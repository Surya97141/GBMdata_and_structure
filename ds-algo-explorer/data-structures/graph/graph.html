Graph

Vertices and edges — directed/undirected, weighted/unweighted. Basis of all graph algorithms.

A graph G=(V,E) has a set of vertices V and edges E. Directed graphs have one-way edges; undirected have two-way. Edges may carry weights. Representations: adjacency matrix (O(V²) space), adjacency list (O(V+E) space), or edge list. The choice affects algorithm complexity significantly.

Purple = highlighted vertex. Orange = highlighted edge. Green = vertex in focus. Each step explores the adjacency list of one vertex.

Space (adj list)
O(V+E)
Space (adj matrix)
O(V²)
Edge check (list)
O(degree)
Edge check (matrix)
