Adjacency List

Array of linked lists — O(V+E) space, O(degree) neighbor scan. Best for sparse graphs.

An adjacency list stores an array of V lists. Each list contains the neighbors (and weights) of that vertex. Space is O(V+E) — only existing edges are stored. Finding neighbors is O(degree) — ideal for sparse graphs where degree ≪ V. Used by BFS, DFS, Dijkstra, and most graph algorithms.

Yellow = currently active vertex row. Purple = current neighbor being visited. Arrows show the linked list structure.
